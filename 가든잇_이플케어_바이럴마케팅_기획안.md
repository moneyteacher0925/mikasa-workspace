## 1. 인스타그램 바이럴 전략

### 1-1. 타겟 인플루언서 유형

| 티어 | 팔로워 | 용도 | 예상 단가(건당) |
|------|--------|------|-----------------|
| 나노 (1K~5K) | 식물 일상 계정 | 시딩 체험단 (무상제공) | 제품만 (0원) |
| 마이크로 (5K~30K) | 플랜테리어/홈가드닝 | 릴스 콘텐츠 제작 | 10~30만원 |
| 미드 (30K~100K) | 반려식물/라이프스타일 | 메인 바이럴 + 스토리 | 50~100만원 |

**탐색 키워드:** #플랜테리어 #식물스타그램 #반려식물 #홈가드닝 #식집사 #식물키우기

### 1-2. 릴스 콘텐츠 포맷 (핵심)

1. **비포/애프터 (가장 강력)**
   - 시든 식물 → 이플케어 분사 → 7일 후 회복 타임랩스
   - 캡션: "포기하려던 몬스테라가 살아났다"
   - 릴스 15~30초, 자막 필수

2. **사용법 ASMR**
   - 미스트 분사 소리 + 잎에 맺히는 물방울 클로즈업
   - 감성적 BGM + "식물도 스킨케어가 필요해요" 자막

3. **식물 응급실 시리즈**
   - "이 식물 살릴 수 있을까?" → 이플케어 처방 → 경과 기록
   - 시리즈물로 팔로워 유지

4. **비교 콘텐츠**
   - 일반 물 vs 이플케어 미스트 → 같은 식물 2주 비교
   - 데이터 기반 신뢰도 ↑

### 1-3. 해시태그 전략 (3단계 조합)

```
[대형] #식물스타그램 #플랜테리어 #반려식물 #홈가드닝 (100만+ 게시물)
[중형] #식물영양제 #식물키우기팁 #식집사일상 #식물미스트 (1~10만)
[브랜드] #가든잇 #이플케어 #2세대식물영양제 #엽면흡수 (신규생성)
```

- 릴스에는 해시태그 5~8개가 최적 (과다 사용 시 알고리즘 불이익)
- **저장·공유 유도 CTA**가 해시태그보다 중요 ("식물 살리는 법 저장해두세요 🔖")

### 1-4. 협찬/시딩 방식

- **1차 시딩 (0원~저비용):** 나노 인플루언서 30~50명에게 제품 무상 제공 → 자율 포스팅
  - 원가 1,600원 × 50명 = **8만원** (배송비 별도)
  - 조건: 인스타 피드/릴스 1건 + 해시태그 3개 필수
- **2차 유료 협찬:** 반응 좋은 계정 중 마이크로 5~10명 선별 → 릴스 제작 의뢰
- **시딩 플랫폼 활용:** 레뷰, 미블, 서울오빠 등 체험단 플랫폼

## 2. 스레드(Threads) 바이럴 전략

### 2-1. 플랫폼 현황

- 2025년 기준 한국 MAU 692만, X(트위터)를 추월
- 텍스트 중심이지만 이미지 첨부 시 반응률 ↑
- **알고리즘 특성:** 팔로잉 기반이 아닌 추천 피드 중심 → 신규 계정도 노출 가능

### 2-2. 바이럴 콘텐츠 유형

1. **TMI/꿀팁형 (가장 효과적)**
   - "식물 영양제 뿌리에 주면 안 되는 경우 알려드림"
   - "엽면 흡수라는 개념 처음 알았는데 식물 살았음"
   - 자연스러운 정보 공유 → 제품 언급

2. **실패→성공 스토리**
   - "식물 7번째 죽이고 있었는데... 이거 쓰고 달라짐"
   - 공감 유도 + 제품 자연 노출

3. **논쟁/질문형**
   - "식물 영양제 1세대 vs 2세대 차이 아는 사람?"
   - 댓글 유도 → 알고리즘 부스트

4. **비교/리뷰형**
   - 다른 영양제 사용 후기 대비 이플케어 차별점

### 2-3. 다계정 운영 전략

- **페르소나별 3~5개 계정 운영:**
  - A계정: 초보 식집사 (시행착오 콘텐츠)
  - B계정: 식물 전문가 (정보/팁 중심)
  - C계정: 플랜테리어 (감성/인테리어)
- **운영 원칙:**
  - 계정 간 자연스러운 상호작용 (리포스트, 댓글)
  - IP/디바이스 분리 필수
  - 각 계정 주 3~5회 포스팅
  - 직접 광고 느낌 배제, 자연스러운 경험담 위주
- **비용:** 운영 인력 1명 + 월 50~80만원 (계정 관리비)

## 3. 유튜브 바이럴 전략

### 3-1. 소형 유튜버(1~5만) 협찬 전략

**이미 확보된 후보:**
| 채널 | 예상 구독자 | 콘텐츠 특성 | 협찬 적합도 |
|------|-----------|------------|-----------|
| 초식남 | 1~3만 추정 | 식물 초보 관점, 친근 | ⭐⭐⭐⭐⭐ |
| 식물튜브/식물아빠 | 1~5만 추정 | 육아+식물 병행, 공감 | ⭐⭐⭐⭐ |
| 그린썸 | 1~5만 추정 | 전문성 있는 가드닝 | ⭐⭐⭐⭐ |

**협찬 단가 참고:**
- 구독자 1만~5만: 건당 **25~50만원** (현금) 또는 제품+수익쉐어
- 니치 분야(식물)는 단가가 일반 대비 낮은 편 → 협상 여지 큼

**접근 방법:**
1. DM/이메일로 제품 소개 + 샘플 발송 제안
2. "뿌리 손상 식물 회복" 킬링포인트 강조 → 콘텐츠 소재로 매력적
3. 영상 기획안 제안 (강제X, 방향성만)

### 3-2. 쇼츠 활용 방안

- **쇼츠 전용 콘텐츠:**
  - "죽어가는 식물 살리는 법 15초" → 미스트 분사 → 회복 타임랩스
  - "식물 영양제 이렇게 쓰면 안 됩니다" → 올바른 엽면 흡수 방법
  - "100원으로 식물 살리기" (1회 사용 원가 환산)
- 쇼츠는 구독자 수 무관하게 알고리즘 노출 → ROI 높음
- 유튜버 협찬 시 "본편 1개 + 쇼츠 1개" 패키지 요청

### 3-3. 제품+수익쉐어 협찬 모델

**구조:**
```
기본: 제품 무상 제공 + 현금 X (또는 소액 10~20만원)
추가: 전용 할인 코드 발급 → 판매 건당 수익 쉐어
```

**수익쉐어 설계 예시:**
- 출시기념가 12,400원 기준
- 유튜버 전용 코드로 10% 추가 할인 (11,160원)
- **판매 건당 1,000~2,000원** 유튜버에게 지급
- 원가 1,600원 기준 마진 충분 (11,160 - 1,600 - 2,000 = 7,560원 마진)

**장점:**
- 초기 현금 지출 최소화
- 유튜버가 적극적으로 홍보할 인센티브
- 성과 측정 용이 (코드별 추적)
