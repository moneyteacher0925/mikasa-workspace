# 딱모여 호스트 등급 구조

> 작성일: 2026-03-25 | 작성: 미카사 | 검토: 에렌, 오맷돌 대표님

---

## 개요

딱모여 모임 운영자(호스트)를 3개 등급으로 구분하여 운영한다.
등급별로 딱모여의 지원 범위와 정산 구조가 다르며, 상위 등급일수록 딱모여의 지원이 두꺼워지고 호스트의 수익 기회도 확대된다.

---

## 1. 호스트 (일반)

### 대상
- 딱모여 플랫폼에서 자유롭게 모임을 주최하고 싶은 개인 누구나

### 운영 방식
| 항목 | 담당 |
|------|------|
| 모임 기획 | 호스트 |
| 장소 섭외 | 호스트 |
| 식사/음료/준비물 | 호스트 |
| 인원 모집 | 호스트 |
| 모임 진행 | 호스트 |
| 결제/정산 시스템 | 딱모여 |

### 정산 구조
- 참가비는 딱모여 계좌로 입금
- 딱모여 수수료 차감 후 호스트에게 정산
- 정산 주기: (협의 필요)

### 수수료율
| 항목 | 비율 |
|------|------|
| 딱모여 수수료 | __% |
| 호스트 정산 | __% |

> 💬 벤치마크: 문토 20%, 소모임 15~20%

### 특징
- 진입장벽 낮음 (누구나 가능)
- 자율성 높음 (기획~운영 전부 본인)
- 딱모여는 플랫폼+결제 인프라만 제공

---

## 2. 슈퍼호스트 (딱모여 직영)

### 대상
- 사람을 모을 수 있는 능력이 있는 사람
- 모임으로 본격적으로 수익을 내고 싶은 사람
- 딱모여와 계약 체결 필요 (계약기간: 1년)

### 운영 방식
| 항목 | 담당 |
|------|------|
| 모임 기획 | 슈퍼호스트 + 딱모여 협의 |
| 장소 섭외 | **딱모여** |
| 식사/음료/준비물 | **딱모여** |
| 인원 모집 (광고/마케팅) | **딱모여** |
| 인원 모집 (자체) | 슈퍼호스트 |
| 모임 진행 | 슈퍼호스트 |
| 결제/정산 시스템 | 딱모여 |

### 정산 구조

**A. 모임 참가비 정산**

| 인원 모집 주체 | 슈퍼호스트 정산 | 딱모여 |
|--------------|---------------|--------|
| 딱모여가 모집 | __% | __% |
| 슈퍼호스트 자체 모집 | __% (더 높음) | __% |

**B. RS 제휴파트너 추가 수익**

- 오프라인 모임에서 딱모여 RS 제휴파트너(보험, 강의, 결정사 등) 연결로 추가 수익 발생 시
- 슈퍼호스트에게 일정 % 추가 정산

| 항목 | 슈퍼호스트 | 딱모여 |
|------|----------|--------|
| RS 추가 수익 | __% | __% |

### 딱모여 지원 내용
1. 모임 장소 섭외 및 예약
2. 식사/음료 등 모임 운영에 필요한 것 전반 처리
3. 광고/마케팅을 통한 모임 인원 모집
4. 정산 및 CS 처리

### 슈퍼호스트 의무
1. 모임 진행 및 참가자 관리
2. 모임 품질 유지 (노쇼/클레임 관리)
3. 계약 기간 내 최소 모임 횟수: (협의 필요)
4. 딱모여 브랜드 가이드 준수

### 계약 조건
- 계약 기간: 1년 (자동갱신 여부 협의)
- 해지 조건: (협의 필요 — 노쇼율, 클레임, 최소 활동 기준 등)

### 특징
- 번거로운 운영은 딱모여가 전담, 호스트는 모임 진행에만 집중
- 이중 수익 구조 (참가비 + RS 추가 수익)
- 자체 모집 능력이 있을수록 더 높은 수익

## 3. VIP 호스트 (프리미엄)

### 대상
- 슈퍼호스트 중 성과가 뛰어나거나 퍼스널 브랜딩 가치가 높은 사람
- 딱모여와 별도 계약 체결

### 운영 방식
- 슈퍼호스트의 모든 지원 포함
- **추가 지원:**

| 항목 | 내용 |
|------|------|
| 영상 제작 | 딱모여가 유튜브/숏폼 소재 제작 |
| 광고 집행 | 딱모여가 광고비 투자하여 집행 |
| 퍼스널 브랜딩 | VIP 호스트 개인 브랜드 구축 지원 |

### 정산 구조
- 슈퍼호스트보다 높은 정산 %
- RS 추가 수익 정산 %도 상향
- 구체적 수치: **추후 협의**

### 특징
- 딱모여가 호스트에게 투자하는 구조
- 호스트의 퍼스널 브랜드 = 딱모여의 트래픽 자산
- 양방향 윈윈: 호스트는 브랜딩, 딱모여는 트래픽
