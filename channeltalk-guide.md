## 1. 채널톡 시작하기

### 핵심 기능 요약
- 채팅 상담, CRM, AI 챗봇(ALF), 사내 메신저를 **하나의 툴**로 통합
- 고객과의 커뮤니케이션을 통해 단골 고객을 만드는 것이 핵심 가치
- 웹/모바일 앱/웹 브라우저 모두 지원

### 설정 방법 (4단계)

**Step 1: 회원가입 & 채널 생성**
- [channel.io/ko](https://channel.io/ko)에서 개인 계정으로 가입 가능 (대표 계정 불필요)
- 사업자 정보 없이 브랜드 이름 + 홈페이지 주소만으로 채널 생성 (3초)

**Step 2: 사이트에 설치**
- **자체 제작 사이트**: [채널 설정] → [일반 설정] → [버튼 설치 및 설정]에서 스크립트 복사 후 삽입
- **빌더사 연동**: 카페24, 아임웹, 식스샵, 고도몰, 가비아 퍼스트몰, 메이크샵, 쇼피파이, 워드프레스, 샵바이, 셀러허브 스토어 공식 지원
- **모바일 앱**: iOS SDK, Android SDK, React Native SDK, Webview 지원 (유료 플랜 필요)
- **홈페이지 없는 경우**: [채널 설정] → [채널 프로필] → [연락처 및 위치]에서 내 채널톡 주소(URL) 복사하여 활용

**Step 3: 고객 맞이 설정**
- **홈 화면 꾸미기**: 브랜드 로고, 대표 색상, SNS 연동
- **봇 인사말 설정**: 봇 프로필과 인사말을 브랜드에 맞게 변경
- **운영 시간 설정**: 고객 응대 가능 시간대 입력 (점심시간 분리 가능)
- **팀 멤버 프로필**: 개인 프로필 사용 권장 (오프라인 점원 느낌), 또는 채널 프로필로 일관된 브랜드 이미지 유지

**Step 4: 데스크탑/모바일 앱 설치**
- [channel.io/ko/download](https://channel.io/ko/download) 또는 웹 [desk.channel.io](https://desk.channel.io)

### 회원 정보 연동
- 홈페이지의 회원 정보를 채널톡에 연동하여 CRM에 활용 가능
- 개발자 문서: https://developers.channel.io/reference/web-quickstart-kr

### 활용 팁
- 튜토리얼 완료 후 **14일 무료 체험** 가능
- 설치 후 '고객과 첫 만남' 설정까지는 반드시 완료 권장

### 주의사항
- 물리망분리/VDI 환경: `*.channel.io`, `*.channel.app` 등 도메인 및 포트 허용 필요
- 네이버 스마트스토어, 지마켓 등 오픈마켓에는 버튼 설치 불가 → 내 채널톡 주소 링크로 우회
- Mobile SDK는 **유료 플랜에서만** 사용 가능
- 허용해야 할 Egress IP: `52.78.175.119/32`, `3.37.79.1/32`, `15.165.26.1/32`, `13.125.193.149/32`

## 2. 채팅 상담 (고객 메신저)

### 핵심 기능 요약
- **상담 구조**: 담당자 1명 + 팔로워(보조 인원) 체계
- **수신함**: 상담 상태별 관리 (진행 중/보류/종료), 필터링, 벌크 액션
- **대화 검색**: 상담 내역 검색 및 일괄 상태 적용
- **입력폼(Form)**: 잠재 고객 정보 획득 → 고객 연락처 저장
- **팔로업 알림**: 고객이 사이트를 떠나도 문자(카카오 알림톡)/이메일로 상담 이어가기
- **멀티 채널 응대**: 여러 채널의 상담을 한 번에 관리
- **상담 프로필**: 상담 관련 정보를 프로필값으로 관리, 필터링/개인화에 활용
- **상담 태그**: 상담 분류 → 통계 분석

### 설정 방법
1. **팀 멤버 초대**: 모든 직원이 고객 소리를 듣도록 초대
2. **담당자 배정**: 채팅마다 담당자 1명 지정, 필요 시 팔로워 추가
3. **매크로 등록**: [수신함] → 자주 쓰는 답변을 매크로로 등록하여 빠르고 일관된 응대
4. **팔로업 알림 설정**: 고객이 사이트를 떠나도 알림 발송 → 미가입 고객도 연락처 저장
5. **상담 종료 관리**: 상담 완료 시 종료 처리, 필요 시 '종료된 상담 답변 막기' 활성화
6. **수신함 알림 설정**: 10분 이상 미배정 상담 알림

### 외부 메신저 통합
- 카카오톡(상담톡), 네이버 톡톡, 라인, 인스타그램 메신저, 이메일 수신함 연동
- 미트(인터넷 전화): 음성/영상 통화 지원

### 활용 팁
- 상담 태그를 활용해 문의 유형 분류 → [분석] → [상담 통계]에서 트렌드 파악
- 팔로워 기능으로 내부 팀원과 실시간 협업 (내부 대화 가능)
- 고객 세그먼트별 차등 응대: VIP는 즉각 응대, 신규 고객은 ALF 자동 응대

### 주의사항
- 종료된 상담에 고객이 재답변하면 상담이 다시 열림 → 필요 시 '종료된 상담 답변 막기' 설정
- 상담 담당자/관리 기능은 **오퍼레이터 시트**가 있는 멤버만 가능

## 3. CRM (고객 연락처)

### 핵심 기능 요약
- **고객 구분**: 리드(비회원)/멤버(회원)/VIP 등 자동 분류
- **고객 프로필**: 홈페이지 회원 정보 연동, 커스텀 프로필 필드 생성 가능
- **고객 연락처**: 엑셀보다 간편한 고객 관리, 검색/필터/세그먼트 저장
- **고객 태그**: 목적별 고객 분류 (예: VIP, 이탈 위험 등)
- **이벤트**: 웹/앱에서의 고객 행동 데이터 추적 (페이지 조회, 결제 완료 등)
- **유니피케이션**: 동일 고객의 정보를 자동 통합 (기기/채널 간)

### 설정 방법
1. **회원 정보 연동 스크립트** 적용 (개발자 문서 참조)
2. **고급 필터**로 원하는 조건 검색 → **세그먼트 저장**
3. **고객 태그** 생성하여 고객 분류
4. 이벤트 데이터 연동 (빌더사 자동 제공 또는 커스텀 이벤트 설정)

### 활용 팁
- 세그먼트는 **필터 조건을 저장**하는 것 → 시간에 따라 대상 고객이 자동 변동
- 세그먼트를 마케팅 캠페인 대상으로 활용
- 객단가 기준으로 고객을 나누어 차등 서비스 제공
- 고객 설문/인터뷰를 통해 단골의 특성을 정의하고 세그먼트화

### 주의사항
- MU(Managed User) = 고객 연락처에 저장된 전체 고객 수 → 요금에 직접 영향
- 고객 연락처에서 삭제 가능하므로 비용 관리 가능
- 무료 플랜: 상담 시작 후 30일간 대화 내역 조회, 이벤트 7일 조회
- 유료 플랜: 기간 제한 없는 상담 검색, 이벤트 90일 조회

## 4. ALF (AI 상담 에이전트)

### 핵심 기능 요약
**고객 ALF v2** — AI가 고객 문의를 자동 처리하는 상담 자동화 기능

- **규칙(Rules)**: 팀 멤버가 미리 작성하는 지시문(프롬프트). 고객 조건/상황에 따라 응대 흐름 제어, 필요 시에만 상담원 연결
- **지식(Knowledge)**: ALF가 참조하는 모든 자료(FAQ, 문서, 파일). 정확하고 일관된 답변 제공
- **태스크(Tasks)**: 주문 취소, 반품 접수, 예약 변경 등 **실제 업무를 AI가 직접 처리**
- **배포 설정**: ALF 프로필, 규칙·지식·태스크 점검 및 관리
- **통계**: ALF의 자동 해결률 모니터링
- **ALF 대화 목록**: ALF가 응대한 상담 일괄 확인
- **ALF 전화 상담 (Beta)**: 상담원 대신 ALF가 전화 응대
- **팀 ALF**: 유저챗/팀챗에서 내부적으로 ALF를 호출하여 업무 활용

### AI 부가기능
- 상담 종료/보류 시 AI가 상담 내용 자동 요약 → 후처리 작업 간편화

### 설정 방법
1. **지식 등록**: FAQ, 도큐먼트, 파일 등 ALF가 참조할 자료 업로드
2. **규칙 작성**: 상황별 프롬프트 작성 (예: "배송 관련 문의 시 ~로 안내")
3. **태스크 설정**: 자동화할 업무 정의 (API 연동 등)
4. **배포 설정**: 프로필, 규칙·지식·태스크 최종 점검 후 배포
5. **통계 모니터링**: 자동 해결률 확인 및 개선

### 활용 팁
- 단순/반복 문의는 ALF가 처리, 복잡한 문의는 상담원 연결로 효율화
- 도큐먼트 기능과 연동하면 ALF가 자동으로 학습하여 응대
- 워크플로우에서 ALF를 호출하여 상담 시나리오에 통합 가능
- VIP 고객은 즉시 상담원 연결, 일반 고객은 ALF 우선 응대 등 규칙 설정

### 주의사항
- ALF 사용량은 **AU(ALF Unit)** 기반 과금 (부가 서비스)
- 무료 체험 시 약 20건(10,000AU)까지 상담 참여 가능
- ALF v2 전용 IP 허용 필요: `3.38.119.225`
- ALF 문제 해결 가이드 제공: 엉뚱한 답변, 지식 미참조, 답변 중단 등 대응 가능

## 5. 워크플로우

### 핵심 기능 요약
- 고객별/상황별 **맞춤형 시나리오를 레고처럼 조립**하여 상담 시작~끝 자동화
- **모듈형 워크플로우**: 트리거 없이 생성, 다른 워크플로우에서 불러오거나 상담 중 호출 가능
- 마케팅으로 전송하여 고객 정보 획득 또는 FAQ 안내 가능

### 하위 카테고리
| 카테고리 | 설명 |
|---------|------|
| 워크플로우란? | 기본 개념 & 핵심 구성 요소 (3개 아티클) |
| 워크플로우 설정하기 | 생성 및 설정 방법 (4개 아티클) |
| 워크플로우 활용 예시 | 상담 여정별 레시피 (상담 시작 전 → 진행 중 → 종료 후) |
| 워크플로우 통계 | 자동 해결률, 상담 인입 분석, 데이터 다운로드 |
| 워크플로우 FAQ | 자주 묻는 질문 모음 |
| 문제 해결 가이드 | 퍼블리시 오류 해결 방법 |

### 설정 방법
1. [워크플로우] → [+ 워크플로우 만들기]에서 레시피 선택 (채팅/전화별 구분)
2. 자동 메시지, 태그 등을 서비스에 맞게 수정
3. 단계별 액션 조립 (조건 분기, ALF 호출, 담당자 배정 등)
4. **퍼블리시**하여 활성화

### 활용 팁
- 준비된 **레시피(템플릿)**를 원클릭으로 적용 후 커스터마이즈
- 중복 시나리오는 모듈형 워크플로우로 만들어 여러 곳에서 재사용 & 동기화
- 상담 시작 전 고객 정보 선수집 → 상담 효율 향상
- 워크플로우 통계로 자동 해결 vs 상담 인입 비율 분석하여 개선

### 주의사항
- 워크플로우는 **부가 서비스** (유료 플랜 필요)
- 무료 제공: 월 50건 실행, 초과 시 실행 건당 과금
- 무료 체험 시 2,000건까지 실행 가능
- 퍼블리시 오류 시 [더보기(︙)] → [오류 단계 보기]로 확인 후 수정

## 6. 마케팅

### 핵심 기능 요약
- **고객에게 먼저 말을 거는 기능** — CRM 데이터 기반 마케팅
- 채널톡 메시지 / 문자 / 이메일 / 카카오 알림톡 / 브랜드 메시지(친구톡)로 전송

### 메시지 유형

| 유형 | 설명 |
|------|------|
| **캠페인** | 조건 + 타이밍에 따라 **자동** 전송. 회원가입 유도, 장바구니 알림 등 |
| **일회성 메시지** | 특정 대상에게 **수동** 발송. 공지, 이벤트, 신제품 출시 등 |
| **알림톡** | 카카오 알림톡 자동 전송. 배송/결제/주문 알림, 리마인드 등 |
| **브랜드 메시지(친구톡)** | 광고성 카카오톡 메시지 전송 |

### 부가 기능
- **마케팅 통계**: 클릭 → 목표달성까지 단계별 유입 경로 분석
- **이벤트 정보**: 고객 행동 데이터 관리, 이벤트명 수정/숨김 가능
- **GTM 연동**: Google Tag Manager 설치로 다양한 마케팅 메시지 설정
- **정보성 vs 광고성 메시지** 구분 관리

### 설정 방법

**캠페인 만들기:**
1. 메시지 입력 (말풍선 버블 또는 전체화면)
2. 대상 고객 설정 (전체 or 세그먼트 불러오기)
3. 시작 이벤트 선택 (예: 페이지 조회, 결제 완료 등)
4. 대기 시간 설정 (즉시 ~ N일)
5. 운영 → [반복 운영] 켜기 (반복 노출 원할 시)

**일회성 메시지:**
1. 메시지 입력
2. 대상 고객: 기본 [휴대폰 번호 있는 고객] + 세그먼트/조건 추가
3. 광고성 메시지 포함 시 반드시 체크 → (광고) 표시 + 수신 거부 링크 자동 삽입

### 활용 팁
- **구매 후 30일 재방문 유도**: 시작 이벤트 [결제 완료] + 대기 30일 + 추가 결제 미발생 필터
- **상세 페이지 체류 고객**: 오래 머무는 고객에게 먼저 말 걸어 상담 유도
- **알림톡 레시피**: 신규가입 혜택 안내, 배송 알림 등 채널톡 제공 템플릿 활용
- 세그먼트 + 캠페인 조합으로 정밀 타겟팅

### 주의사항
- 마케팅은 **부가 서비스** (유료 플랜 필요)
- 무료 제공: 월 30건 실행, 초과 시 건당 과금
- 문자/알림톡 발송 시 **PU(Post Unit)** 실비 비용 발생
- 광고성 메시지는 반드시 (광고) 표시 필수
- 무료 플랜: 카카오/문자 월 10,000PU 한도
- PU 선불 충전: [구독 및 결제] → [사용량 선불 충전] → [연동 서비스 충전]

## 7. 팀 메신저

### 핵심 기능 요약
- **고객 상담 + 사내 메신저를 하나의 툴**로 통합한 유일한 솔루션
- 팀원 간 대화하면서 동시에 고객 상담 내역 확인 가능 (사이드뷰)
- **스레드**: 하나의 주제에 대해 체계적으로 대화
- **방해 금지 모드**: 퇴근 후 알림 차단 설정

### 구조
- 팀챗: 팀원 간 대화 공간
- 사이드뷰: 팀 대화 중에도 고객 대화 내역을 한 화면에서 확인
- 스레드: 주제별 대화 분리

### 설정 방법
1. 팀원 초대: [채널 설정] → [팀원 구성]에서 팀원 초대
2. 팀/그룹 생성: 목적에 맞는 팀 형성
3. 역할/권한 설정: 팀원별 역할 및 권한 부여
4. 방해 금지 모드: 개인 설정에서 원하는 시간대 알림 차단

### 활용 팁
- 고객 상담 중 내부 대화로 동료에게 도움 요청 → 별도 앱 전환 불필요
- 여러 창을 켜지 않아도 사이드뷰로 필요한 정보를 한 화면에서 확인
- **무료 플랜에서도 팀 메신저 이용 가능**

### 주의사항
- 기본 시트만으로도 팀챗 이용 + 상담 내역 조회 + 팔로워 참여 가능
- 상담 담당자가 되려면 오퍼레이터 시트 필요

## 8. 요금제/플랜

### 구독 체계 요약

| 구분 | 내용 |
|------|------|
| **무료 플랜** | 팀 메신저 + 고객 상담 (MU/시트 무제한), 외부 메신저 연동 무료 |
| **얼리 스테이지** | 소규모 비즈니스용 유료 플랜 |
| **그로스** | 성장 단계 비즈니스용 |
| **엔터프라이즈** | 대규모 기업용 (보안 패키지 포함) |

### 핵심 개념

**MU (Managed User)**
- 고객 연락처에 저장된 전체 고객 수
- 플랜별 기본 제공량 있음, 초과 시 추가 비용

**시트**
| 시트 유형 | 기능 |
|-----------|------|
| **기본 시트** | 팀챗, 상담 내역 조회, 고객 연락처 열람, 팔로워 참여 |
| **오퍼레이터 시트** | 기본 시트 + 상담 담당자/관리, 전화 기능, 상담 상태 변경/태그 |

**부가 서비스 (유료 플랜 필수)**
| 서비스 | 무료 제공량 | 과금 방식 |
|--------|-----------|-----------|
| 워크플로우 | 월 50건 | 실행 건당 후불 |
| 마케팅 | 월 30건 | 실행 건당 후불 |
| 고객 ALF | AU 기반 | 해결 건당 후불 |
| 보안 패키지 | - | 엔터프라이즈 전용 |
| 화이트 라벨 | - | 채널톡 로고 제거 |

### 결제 방식
- **월구독**: 매월 납부
- **연구독**: 12개월 일시 납부 (월구독 대비 할인)
- **사용량 후불**: 플랜 제공량 초과분 자동 청구
- **선불 충전**: 12개월분 선구매 시 최대 25% 할인 (약 3개월 무료)

### 무료 플랜 상세
- MU/시트 무관하게 팀 메신저 + 고객 상담 가능
- 네이버 톡톡, 인스타그램, 이메일, 라인 연동 무료
- 카카오 상담톡/알림톡/브랜드 메시지 가능 (월 10,000PU 한도)
- 상담 내역: 30일 조회 / 이벤트: 7일 조회
- 팔로업 알림: 월 10,000PU 한도
- 미트: 최대 30명, 60분

### 유료 플랜 추가 기능
- 고객 정보 조회, 상담 내역 무기한 검색
- 이벤트 90일 조회
- 미트(전화) 이용 (오퍼레이터 시트 필요)
- OpenAPI 및 모바일 SDK
- 미트: 최대 200명, 시간 무제한

### 14일 무료 체험
- **경로**: [채널 설정] → [구독 및 결제] → [서비스 구독] → [유료 기능 14일 체험하기]
- 채널 당 1회 제공
- 결제 카드 등록 필요 (체험 중 비용 미발생)
- 체험 종료 다음 날 예약 플랜으로 자동 결제
- **체험 중 제한**: 문자/이메일 신규 상담 불가, OpenAPI 불가, 워크플로우 2,000건/마케팅 1,000건 한도, ALF 약 20건

### 환불 정책
- 전액 환불: 미사용 시 7일 이내 신청
- 부분 환불: 잔여 일수만큼 예치금으로 환불
- 추가 사용량은 환불 불가

### 주의사항
- 이미 사용량으로 집계된 팀 멤버를 삭제해도 시트 사용량 미차감
- 문자/소셜미디어/전화 실비는 선불 충전 할인 미적용
- 플랜 업그레이드/다운그레이드 자유롭게 가능

## 추가 카테고리 (참고)

### 채널 설정
- **상담 설정**: 인사말, 운영 시간, 자동 배정 등 (6개 아티클)
- **일반 설정**: 채팅 버튼, 라운지 디자인 (5개 아티클)
- **팀원 구성**: 팀/팀원 관리, 역할 설정 (3개 아티클)
- **보안 설정**: 이중 인증, 파일 보안
- **보안 패키지**: SAML SSO, IP 주소 제한 (엔터프라이즈)

### 분석
- 상담 통계, 커스텀 리포트 등 9개 아티클

### 외부서비스 연동 (18개 아티클)
- 카카오톡(상담톡), 네이버 톡톡, 라인, 인스타그램, 이메일 등

### 미트 (인터넷 전화, 10개 아티클)
- 일반 전화 수/발신, 음성 미트, 영상 미트

### 도큐먼트 (9개 아티클)
- 사용 가이드, 블로그, 릴리즈 노트 등 문서 관리
- ALF가 도큐먼트 학습하여 고객 응대에 활용

### 앱스토어 (8개 아티클)
- 다양한 커맨드 앱 연동
