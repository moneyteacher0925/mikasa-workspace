에이피알 광고/마케팅 전략

1. 틱톡 어필리에이트 (핵심 무기)
틱톡 크리에이터가 제품 리뷰 → 틱톡샵에서 바로 구매 연결
초기부터 틱톡 어필리에이트에 올인, 구매 전환 직전 노출에 집중
할인 의존 없이 매출+이익 동시 확보하는 구조
2. SNS 인플루언서 "인해전술"
10~30대 여성 타겟 → TV 대신 SNS에 마케팅비 집중
대형 인플루언서 + 마이크로 인플루언서 동시 활용
광고선전비 매출 대비 17% — 전부 디지털에 투입
3. 아마존 내부 광고 (미국)
아마존 스폰서드 광고 + 프라임데이 집중 공략
실구매 직전 노출 광고에 집중하는 퍼포먼스 전략
2025 Q3 미국만 분기 1,500억 돌파
4. D2C (자사몰 직접 판매)
자체 웹사이트에서 직접 판매 → 중간 마진 제거
고객 데이터 직접 수집 → 데이터 기반 마케팅 내재화
리타게팅, 개인화 추천 등 정교한 CRM
5. 오프라인 확장 (최근)
미국 울타(ULTA) 뷰티 입점 → 오프라인 채널 확대
성수동 메디큐브 플래그십 → 오픈 1달 만에 1.5만명 방문
온라인→오프라인 확장으로 브랜드 신뢰도 강화

핵심 인사이트

한국 기업이 미국식 마케팅으로 성공한 케이스.

- TV 광고 X → 틱톡 + 인플루언서에 올인
- 네이버 블로그 마케팅 X → 아마존 내부 광고 + 틱톡샵
- 브랜딩보다 구매 전환 직전 노출에 집중
- 할인으로 매출 뽑지 않고 콘텐츠로 수요 창출
- 결과: 해외 매출 80%, 영업이익률 24%

에렌이 물어본 "한국 vs 미국 마케팅 차이" — 에이피알은
한국 시장은 한국식으로, 해외는 미국식으로 쓰는 하이브리드 전략
