# 그로우팀 미팅 리포트

**일시:** 2026년 4월 8일 (화) 20:00
**참석:** 권현우 (진격의 마케팅) × 그로우팀 대표님
**주제:** 메타 퍼포먼스 광고 컨설팅 및 협업 방향

---

## 1. 미팅 핵심 요약

### 추천 프로그램
- **그로우팀 30일 동행 프로젝트** (메타 광고 노트 + 1:1 온라인 코칭 + 무제한 카톡 Q&A)
- 직접 세팅하고 대표님이 코칭해주는 방식 → 실력 내재화에 최적

### 메타 광고 시작 전 필수사항
1. **메타 픽셀 + 전환 API(CAPI) 설치** — 사이트 활동 트래킹의 기본
2. **페이스북 페이지 생성 후 2~3주 활동** — 메타 보안 정책상 유령계정 차단이 엄격, 활동 이력 필요
3. 활동 후 **비즈니스 페이지(비즈니스 스위트)** 생성
4. 그 다음 광고 세팅 진행

### 사이트별 픽셀/전환API 설치 난이도

| 사이트 | 플랫폼 | 난이도 | 비고 |
|--------|--------|--------|------|
| 진격의 마케팅 (attackmarketing.cafe24.com) | 카페24 | 쉬움 | 카페24 자체 연동 지원, 천주원님 도움 요청 |
| 딱모여 (ddakmoyeo.com) | 버셀 (자체개발) | 복잡 | 전환API 직접 구현 필요, 개발자 투입 2~3주 |

## 2. 액션플랜 (실행 순서)

### STEP 0: 픽셀 & 전환API 설치 (선행 필수)
- **마케팅해줘 (카페24):** 천주원님에게 픽셀 + 전환API 설치 요청
- **딱모여 (버셀):** 개발자 외주 또는 Claude Code로 구현 (2~3주)
- **확인 도구:** Chrome 확장 "Facebook Pixel Helper" 설치 → 녹색 표시 확인

### STEP 1: 페이스북 계정 활성화 (2~3주)
- 메타 비즈니스 스위트 로그인
- 페이스북 페이지 생성 (진격의 마케팅 / 딱모여 각각)
- 2~3주간 꾸준히 활동 (게시물 올리기, 좋아요, 댓글 등)
- 목적: 메타 보안 시스템에서 정상 계정으로 인식시키기

### STEP 2: 비즈니스 페이지 세팅
- 메타 비즈니스 관리자 설정
- 광고 계정 생성
- 결제 수단 등록

### STEP 3: 동행 프로그램 결제 & 광고 세팅
- 그로우팀 30일 동행 프로젝트 결제
- 직접 광고 세팅 → 대표님 코칭/피드백
- 캠페인 구조: 트래픽 → 전환 순서로 테스트

### STEP 4: 운영 & 최적화
- 그로우팀과 지속 소통하며 광고 최적화
- 소재 A/B 테스트, 타겟 조정
- ROAS/CPA 모니터링

## 3. 픽셀 & 전환API 설치 외주 비용 (시세)

| 구분 | 비용 | 비고 |
|------|------|------|
| 카페24 픽셀+전환API | 10~30만원 | 솔루션 자체 연동이라 간단 |
| 자체개발 사이트 (버셀 등) | 30~80만원 | GTM 활용 or 서버사이드 직접 구현 |
| 크몽 기준 픽셀 설치 | 10~50만원 | 고도화 수준에 따라 상이 |
| 컨설팅 포함 (진단+설치+교육) | 50~100만원 | 약 2주 소요, 미팅 포함 |

> 천주원님이 마케팅해줘 쪽을 도와줄 수 있으면 카페24는 비용 절감 가능.
> 딱모여(버셀)는 Claude Code로 직접 구현하면 외주비 0원, 단 2~3주 공수.

## 5. 참고사항
- 메타는 2025년부터 픽셀 + 전환API **병행 설치**를 공식 권장
- 픽셀만으로는 iOS 개인정보 보호 정책(ATT)으로 데이터 누락 발생
- 전환API는 서버에서 직접 데이터를 보내므로 브라우저 차단 우회 가능
- **픽셀 헬퍼** (Chrome 확장) 설치해서 트래킹 정상 작동 항상 확인
