# 보험설계사 자격 취득 가이드
> 작성: 2026-04-07 | 목적: 딱모여 RS 구조 내 보험 연결 수수료 코드 확보

---

## 1. 왜 필요한가

- 딱모여 모임 → 보험 니즈 발생 → 계약 연결 시 수수료(RS) 수취
- 보험설계사 코드가 있어야 계약 체결 중개 가능
- 에렌 개인 명의로 취득 후 GA 소속 등록

---

## 2. 결격사유 체크 ✅

| 항목 | 해당 여부 |
|------|----------|
| 미성년자·피성년후견인 | ❌ 해당없음 |
| 파산선고 후 미복권 | ❌ 해당없음 |
| 금고 이상 실형 (집행 후 3년) | ❌ 해당없음 |
| 금고 이상 집행유예 (유예기간 중) | ❌ 해당없음 |
| 보험업법 위반 벌금 (3년 이내) | ❌ 해당없음 |
| 설계사/대리점 등록 취소 (2년 이내) | ❌ 해당없음 |

> 과거 벌금 300만원은 보험업법 위반이 아니므로 결격사유 없음.
> 근거: 보험업법 제84조 제2항

---

## 3. 취득 절차 (총 3~4주)

### STEP 1: 보험연수원 교육 이수 (1~2주)

| 항목 | 내용 |
|------|------|
| 사이트 | https://www.in.or.kr |
| 교육 형태 | 온라인 (PC/모바일) |
| 교육 시간 | 약 40시간 |
| 교육비 | 약 3~6만원 |
| 수료 조건 | 전 과정 수강 + 진도율 충족 |

**해야 할 것:**
1. 보험연수원 회원가입
2. 보험설계사 등록교육 신청 (생명/손해/제3보험 중 선택)
3. 온라인 강의 수강 → 수료

> 💡 딱모여 RS 구조상 손해보험+제3보험 통합이 범위가 넓어서 유리

---

### STEP 2: 자격시험 응시 (준비 1~2주)

| 항목 | 내용 |
|------|------|
| 주관 | 보험연수원 |
| 시험 형태 | 사지선다 객관식 |
| 문항 수 | 단일: 45문항 / 통합: 60문항 |
| 시험 시간 | 단일: 50분 / 통합: 70분 |
| 합격 기준 | 100점 만점 60점 이상 |
| 난이도 | ★★☆☆☆ (1~2주 준비면 충분) |
| 합격률 | 약 70~80% |
| 응시료 | 약 1만원 |

**시험 과목:**
- 공통 (20문항): 보험 기초지식, 모집 법규
- 전공 (25문항): 선택한 보험 종목별 약관·상품
- 제3보험 (15문항): 상해/질병/간병보험

**공부 방법:**
1. 보험연수원 교재 (교육 신청 시 제공)
2. 교재 1회독 + 핵심 정리
3. 기출 유사 문제 풀이 (보험연수원 모의고사)
4. 오답 정리 → 재시험 시 60점 여유있게 통과

> ⚠️ 기출문제 비공개 (문제지+답안 회수). 교재 중심으로 공부할 것

---

### STEP 3: GA 소속 등록 → 코드 발급 (1주)

| 항목 | 내용 |
|------|------|
| GA란 | 법인보험대리점 (여러 보험사 상품 취급) |
| 등록 방법 | GA 방문 → 위촉 계약 → 보험협회 등록 |
| 코드 발급 | 등록 완료 후 보험사별 모집인 코드 부여 |
| 소요 기간 | 약 3~7일 |

**GA 선택 기준:**
- 수수료율 높은 곳 (GA마다 다름, 비교 필수)
- 정착지원금 안 받기 (환수금 리스크 제거)
- 자유도 높은 곳 (출근 의무 없는 GA)
- 취급 보험사 수 많은 곳

**대형 GA 참고:**
- 피플라이프, 리치앤코, GA코리아, 보험클리닉 등
- 미팅 후 조건 비교하고 결정

---

## 4. 전체 타임라인

```
Week 1-2  보험연수원 온라인 교육 이수
Week 2-3  시험 준비 (교재 1회독 + 문제풀이)
Week 3    시험 응시 → 합격
Week 4    GA 선택 → 위촉 계약 → 코드 발급
```

**예상 총 비용:** 약 7~10만원 (교육비 + 응시료)

---

## 5. 코드 발급 후 활용

- 딱모여 모임 참가자 중 보험 니즈 발생 시 상담·계약 연결
- 계약 체결 → 보험사에서 GA로 수수료 지급 → GA에서 에렌에게 배분
- 수입은 에렌 개인 사업소득으로 잡힘

**법인 매출로 전환하려면:**
- 향후 플로우머스 명의 법인보험대리점 등록 검토
- 유자격자 1명(에렌) 있으면 등록 가능

---

## 6. 주의사항

- [ ] 정착지원금 받지 않기 (환수금 발생 방지)
- [ ] GA 계약 시 수수료율·조건 서면 확인
- [ ] 보험 모집 시 설명의무·적합성 원칙 준수 (법적 의무)
- [ ] 자격증 유효기간 없음 (등록 유지만 하면 됨)

---

- [ ] 온라인 교육 이수
- [ ] 자격시험 접수

- [ ] 보험협회 등록 → 코드 발급
