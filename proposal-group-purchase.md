### 1. 회사 소개

**가든잇(주)**는 "건강한 흙에서 건강한 삶이 시작된다"는 철학 아래, 안전하고 품질 높은 원예 자재를 제공하는 기업입니다.

- **설립:** [설립연도]
- **주요 사업:** 무농약 원예 자재 제조·유통
- **비전:** 모든 가정에 안전한 가드닝 경험을 제공

---

### 2. 제품 소개: 무농약상토

| 항목 | 내용 |
|------|------|
| 제품명 | 가든잇 무농약상토 |
| 용량 | 10L / 20L / 50L |
| 특징 | 화학 농약 Zero, 유기물 기반 프리미엄 배양토 |
| 인증 | 무농약 인증 / 친환경 원료 사용 |
| 소비자가 | [가격대 기재] |

**왜 구독자가 좋아할까?**
- 아이·반려동물 안전 — 무농약이라 가족 모두 안심
- 초보자도 실패 없는 최적 배합
- 채소, 허브, 관엽식물, 다육 등 올라운드 사용

---

### 3. 공동구매 구조

```
┌─────────────┐    콘텐츠 & 구매링크     ┌──────────┐
│ 크리에이터님  │ ──────────────────────▶ │  구독자   │
│  (채널 운영)  │                        │  (구매자) │
└──────┬──────┘                        └─────┬────┘
       │                                      │
       │  수익 배분 (RS)                        │ 주문
       │                                      ▼
       │                              ┌──────────────┐
       ◀──────────────────────────── │  가든잇(주)    │
                                      │ 재고·배송·CS  │
                                      └──────────────┘
```

**진행 방식:**
1. 크리에이터님 전용 **공동구매 링크(또는 할인코드)** 발급
2. 크리에이터님이 자유롭게 콘텐츠 제작 & 공동구매 안내
3. 구독자가 전용 링크로 주문
4. 가든잇(주)에서 **재고 관리, 배송, 고객 응대 전담**
5. 정산 주기에 따라 **판매 수익 배분**

---

### 4. 수익 배분 조건

| 항목 | 내용 |
|------|------|
| **배분 비율** | 판매가의 **10~20%** (구독자 규모·판매량에 따라 협의) |
| **정산 주기** | 월 1회 (익월 15일 지급) |
| **정산 방식** | 계좌이체 / 세금계산서 발행 가능 |
| **최소 보장** | 없음 (리스크 Zero — 판매된 만큼만 정산) |
| **계약 기간** | 1회성 또는 장기 (협의) |
| **추가 인센티브** | 월 판매 목표 초과 시 추가 보너스 협의 가능 |

**💰 수익 시뮬레이션 (예시)**

| 시나리오 | 판매 건수/월 | 객단가 | 배분율 | 월 수익 |
|---------|------------|--------|--------|---------|
| 보수적 | 100건 | 25,000원 | 15% | **375,000원** |
| 기대치 | 300건 | 25,000원 | 15% | **1,125,000원** |
| 낙관적 | 500건 | 25,000원 | 15% | **1,875,000원** |

> 1회성 광고비가 아닌, **매월 반복되는 패시브 인컴**입니다.

### 5. 크리에이터 혜택

| 혜택 | 설명 |
|------|------|
| 💰 **지속적 수익** | 콘텐츠가 노출되는 한 계속 수익 발생 (검색·추천 알고리즘 효과) |
| 📦 **운영 부담 Zero** | 재고, 배송, CS, 교환/환불 모두 가든잇(주) 전담 |
| 🎁 **제품 무상 제공** | 리뷰·콘텐츠 제작용 제품 무상 지원 |
| 🏷️ **구독자 전용 혜택** | 공동구매 전용 할인가 제공 → 구독자 만족도 ↑ |
| 📊 **실시간 대시보드** | 판매 현황·수익 실시간 확인 가능 (제공 예정) |
| 🤝 **자유로운 콘텐츠** | 제작 방식·일정 모두 크리에이터님 자율 |
| 🔄 **리스크 Zero** | 초기 투자 없음. 안 팔리면 비용도 없음 |
