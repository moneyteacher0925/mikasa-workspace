## 헤더 (GNB)

```
[로고: 진격의 마케팅]  마케팅 | 프로그램 | 마케팅 대행 | 무료 세미나 | 전문직 마케팅 | 1:1 컨설팅  [문의하기]
```

- 로고 클릭 → 메인
- 문의하기 버튼 → /마케팅대행 문의폼 또는 카카오톡 채널

## 카카오톡 채널 채팅 위젯

- 우측 하단 플로팅 버튼
- "궁금한 점이 있으신가요?" 말풍선

---

## 공통 카피 톤 가이드

- **전문적이되 어렵지 않게** — 업계 전문용어는 최소화
- **실적 기반** — 숫자로 증명 (2,500+, 1,800+, 8년+)
- **행동 유도** — 모든 섹션 끝에 CTA 배치
- **문제 → 해결** 구조 — 고객의 고민을 먼저 언급, 해결책 제시
- 존댓말 (합쇼체 / ~합니다 / ~드립니다)

## 컬러 가이드 (제안)

- Primary: 레드 계열 (#E53935 또는 유사) — "진격" 이미지

- Accent: 화이트 배경 + 레드 CTA 버튼
- 신뢰감을 위해 과도한 색 사용 자제
