# 전문직 마케팅 (B2B, 기획 동반)

## 페이지 구조 — 랜딩페이지형

### 섹션 1: 히어로

**헤드라인:**
> 의사, 변호사, 세무사 — 전문직에는 전문직 마케팅이 필요합니다.

**서브카피:**
> 일반 마케팅과 전문직 마케팅은 다릅니다.
> 광고법 준수, 신뢰 중심 브랜딩, 지역 타겟팅까지.
> 진격의 마케팅이 전문직 특화 전략을 설계합니다.

**CTA:**
[무료 상담 신청하기] ↓ 문의폼으로 스크롤

---

### 섹션 2: 왜 전문직 마케팅은 다른가

| 일반 마케팅 | 전문직 마케팅 |
|------------|-------------|
| 범용 키워드 공략 | 지역 + 전문 분야 키워드 |
| 자극적 카피 OK | 광고법·윤리규정 준수 필수 |
| 단기 트래픽 중심 | 신뢰·전문성 기반 장기 브랜딩 |
| 일회성 프로모션 | 환자/의뢰인 지속 유입 구조 |

---

### 섹션 3: 대상 전문직

- 🏥 **의사/치과/한의원** — 지역 상위노출, 후기 관리, 블로그
- ⚖️ **변호사/법무사** — 전문 분야별 키워드, 신뢰 콘텐츠
- 📊 **세무사/회계사** — 시즌별 마케팅, 지역 타겟
- 🏗️ **건축사/감정평가사** — B2B + B2C 하이브리드 전략
- 기타 전문직 상담 가능

---

### 섹션 4: 서비스 프로세스

```
[1. 무료 상담] → [2. 업종 분석] → [3. 맞춤 전략] → [4. 실행] → [5. 성과 리포트]
```

1. **무료 상담** — 현재 상황, 목표 환자/의뢰인 수 파악
2. **업종 분석** — 해당 전문직 시장, 경쟁 현황, 광고법 확인
3. **맞춤 전략** — 지역 + 전문 분야 최적화 전략 설계
4. **실행** — 블로그, 플레이스, SNS, 광고 통합 운영
5. **성과 리포트** — 월간 신환/신규 의뢰인 데이터 리포팅

---

### 섹션 5: 실적

> **500+** 전문직 프로젝트 수행
> **평균 2.8배** 신환/의뢰인 증가

### 섹션 7: 문의폼 (DB 수집)

**헤드라인:**
> 전문직 마케팅, 무료 상담부터 시작하세요.

**입력 필드:**
- 업종 (필수 — 드롭다운: 의사/치과/한의원/변호사/세무사/회계사/기타)
- 병원·사무소명 (필수)
- 담당자명 (필수)
- 연락처 (필수)
- 이메일
- 지역 (시/구 단위)
- 현재 마케팅 상태 (선택 — 안 하고 있음 / 직접 운영 / 타 대행사)
- 월 예산 (선택 — 100만원 미만 / 100~300만원 / 300~500만원 / 500만원 이상)
- 기타 문의사항 (텍스트)
