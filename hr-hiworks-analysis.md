# 하이웍스(Hiworks) HR 시스템 분석

> 작성일: 2026년 3월 9일 | 대상: 소규모 회사(정직원 4명) 관점

---

## 1. 하이웍스 개요

- **국내 시장 점유율 1위** 클라우드 그룹웨어
- 인사관리, 근무관리, 경비지출관리, 메신저, 전자결재, 세금계산서 등 올인원 솔루션
- 가비아(Gabia) 계열사 운영

## 3. 소규모 회사(정직원 4명)에서의 활용도 평가

### 🟢 잘 맞는 점 (활용도 높음)

1. **무료 근태관리 (10인까지)**
   - 4명이면 완전 무료 범위 내
   - 모바일 GPS 출퇴근, 연차 자동 관리만으로도 상당한 가치
   - 엑셀 수기 관리 대비 큰 효율 향상

2. **기업 메신저 무료**
   - 조직도 기반 메신저로 업무 소통
   - 카카오톡 업무 사용 탈피 가능

3. **전자세금계산서 100건 무료**
   - 소규모 기업에 초기 충분한 양

4. **법인카드 연동 (2개 무료)**
   - 경비 관리 자동화

5. **클라우드 기반**
   - 별도 서버 불필요, 어디서든 접근

### 🟡 보통인 점

6. **전자계약**
   - 건당 1,000P로 비용 발생하나, 근로계약서 등 연 몇 건이면 부담 적음

7. **UI/UX**
   - 기능은 다양하나 초기 설정이 다소 복잡하다는 평가

### 🔴 부족한 점

8. **급여 계산·이체 기능 없음**
   - 급여대장 설정은 유료, 자동 계산/이체는 미지원
   - 4대 보험료 계산, 원천세 계산 등은 별도 필요

9. **성과 관리 기능 없음**
   - OKR, KPI, 피드백 등 성과 관리 도구 미제공

10. **법률 컴플라이언스 가이드 없음**
    - 5인 미만 사업장 법적 요건 체크리스트 등 미제공

## 4. 유료 플랜 비용 분석

| 상품 | 가격 | 비고 |
|------|------|------|
| 인사/회계 CORE | **무료** | 10인까지 근무관리 무료 |
| 근무관리(+전자결재) | 인당 월 요금 (첫해 할인) | 전자결재, 유연근무, 급여대장 등 |
| 세금계산서 정액제 | 월 정액 | 매달 약정 건수만큼 발급 |
| 하이웍스 포인트 | 충전식 | 건별 서비스 이용 시 차감 |

→ 4인 기업이면 **무료 플랜만으로 기본 근태관리 충분**, 전자결재가 꼭 필요하지 않다면 유료 전환 불필요

## 5. 현재 급여 프로세스 분석 (실장님 피드백 반영)

### 현재 프로세스 (AS-IS)

```
하이웍스 근태기록 → 엑셀 다운로드 → 수작업 근무시간 계산 
→ 엑셀을 세무사에게 전달 → 세무사가 급여명세서 입력 
→ 아르민 실장님이 나하고(NAHAGO) 앱으로 각 직원에게 전송
```

### 문제점

| 문제 | 상세 |
|------|------|
| **앱 3개 설치 부담** | 직원이 하이웍스 + 하이웍스 메신저 + 나하고 총 3개 설치 필요 |
| **수작업 병목** | 근태 → 엑셀 → 수작업 계산 구간에서 오류 발생 가능 |
| **프로세스 길이** | 6단계를 거쳐야 급여명세서 전달 완료 |
| **시스템 단절** | 하이웍스(근태) ↔ 세무사(급여) ↔ 나하고(명세서) 3개 시스템이 연동 안 됨 |
| **알바/프리랜서 관리 어려움** | 시즌별 4~5명 알바 + 재택 프리랜서의 근태/급여를 같은 방식으로 관리하기 번거로움 |

### 개선 방향 (TO-BE)

1. **급여 자동계산 솔루션 도입**: 자비스(JOBIS) 또는 페이히어(Payheer)로 하이웍스 근태 데이터 연동 → 급여 자동 계산 → 명세서 자동 발송
2. **앱 통합**: 가능하면 하이웍스 하나로 근태+메신저+급여명세서 확인까지 (유료 전환 검토 또는 자비스 앱으로 명세서 통합)
3. **프리랜서/알바 별도 관리**: 3.3% 원천징수 대상 프리랜서는 별도 간이 프로세스, 알바는 시급 계산 자동화

## 6. 부족한 부분과 보완 방안

| 부족한 영역 | 보완 방안 | 추천 도구 |
|-------------|-----------|-----------|
| **급여 자동계산·이체** | 급여 전문 솔루션 도입 (세무사 위탁 유지하되 수작업 최소화) | 자비스(JOBIS), 페이히어(Payheer) |
| **급여명세서 통합 전달** | 나하고 앱 의존 탈피, 급여솔루션 내 명세서 발송 | 자비스 앱 또는 하이웍스 유료 |
| **성과 관리** | 경량 도구로 자체 운영 | Notion, Google Sheets + OKR 템플릿 |
| **채용 관리** | 소규모는 별도 ATS 불필요 | 잡플래닛, 원티드 등 채용 플랫폼 직접 활용 |
| **교육·학습** | 외부 플랫폼 활용 | 클래스101, 인프런, LinkedIn Learning |
| **법률 컴플라이언스** | 노무사 자문 또는 가이드 구독 | 노무법인 상담, 고용노동부 무료 상담(1350) |
| **전자결재** | 4명이면 메신저/이메일 승인으로 대체 가능 | 하이웍스 메신저 또는 Slack |
| **복리후생 관리** | 별도 관리 | 스프레드시트 or 플렉스(flex) |

## 7. 종합 평가

### 점수 (5점 만점, 4인 기업 관점)

| 항목 | 점수 | 평가 |
|------|------|------|
| 가격 대비 가치 | ⭐⭐⭐⭐⭐ | 무료로 핵심 근태관리 가능 |
| 근태관리 | ⭐⭐⭐⭐ | GPS 출퇴근, 연차 자동관리 우수 |
| 급여관리 | ⭐⭐ | 기본적, 자동계산/이체 미흡 |
| 사용 편의성 | ⭐⭐⭐ | 기능 많으나 초기 설정 복잡 |
| 확장성 | ⭐⭐⭐⭐ | 인원 증가 시 유료 전환 용이 |
| 올인원 완성도 | ⭐⭐⭐ | HR+회계+메신저이나 급여·성과 부족 |

### 결론

> **하이웍스 무료 플랜은 4인 소규모 기업의 근태·인사 관리 출발점으로 매우 적합합니다.** 무료 범위 내에서 출퇴근 체크, 연차 관리, 기업 메신저, 경비관리를 해결할 수 있으며, 별도 서버나 설치 없이 바로 시작 가능합니다.
>
> 다만 **급여 계산·4대보험 신고는 세무사 위탁 또는 자비스 등 전문 솔루션으로 보완**하고, **성과관리는 Notion/Google Sheets로 경량화**하는 조합이 가장 실용적입니다.
>
> 인원이 10명을 넘기 전까지는 무료 플랜을 최대한 활용하고, 이후 전자결재·유연근무 등이 필요해지면 유료 전환을 검토하는 것을 권장합니다.
