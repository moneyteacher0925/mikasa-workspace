⚡ 핵심 원칙

1. 관여도가 낮으면 절대 상세페이지로 바로 보내지 않는다 → 콘텐츠 2~3회 소비 후 전환
2. 관여도가 높으면 빠르게 전환한다 → 로열티 높은 수강생은 CRM → 상페 직행
3. 모든 채널의 종착지는 하나 → 상세페이지 (결제)
4. 유튜브 + 카페/블로그가 관여도 끌어올리는 3대 허브 → 영상만한 게 없다 + 후기/변화 스토리
5. 카카오 팀채팅 = 클로징 공간 → 질문 대응 + 분위기 + 마감 압박
6. 55만원은 충동구매 금액이 아니다 → 최소 3회 접촉 원칙 (신규 기준)
