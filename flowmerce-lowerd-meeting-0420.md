## 2. 주요 기능 안내

### 마일리지 (포인트)
- 마일리지 = 포인트 개념
- **이름 변경 가능** (ex. "포인트", "캐시" 등 자유 설정)

### 클래스 관리
- **클래스 등록** 시 이미지 최적 사이즈: **1280 x 720 px**
- **연장권 설정 가능** — 강의 기간 종료 후 연장권 구매 유도
  - 연장권은 전환율이 있는 편이므로 **설정 권장**
- **오프라인 기능 미지원** — 오프라인 강의 시 결제 랜딩페이지로만 활용 가능

### 프로모션 관리
- 프로모션 설정 기능 제공

### 디자인 관리
- **랜딩페이지 리스트** 에서 관리
- 상단 배너 변경 등 커스터마이징 가능

## 3. 알림톡 운영 꿀팁

### 템플릿 승인 전략
| 카테고리 설정 | 승인율 |
|--------------|--------|
| 서비스이용 / 이용안내 | 높음 |
| 서비스이용 / 리마인드 | 높음 |

- 규제 이슈로 **과도한 후킹 문구는 지양**
- **핵심 트릭:** 템플릿에는 `이름 / 인원수 / 강의링크`를 깔끔하게 구성하고, **강의링크 URL 자체에 후킹 요소를 포함**시켜 전환 유도

## 4. 토스페이먼츠 (결제 연동)

|------|------|
