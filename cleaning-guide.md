# 가든잇 청소 가이드 & 체크리스트

---

## 기본 원칙

- **청소 순서**: 화장실(내부→외부) → 휴게실 → 사무실 (오염도 높은 곳 먼저)
- **장갑 착용 필수** (특히 화장실)
- 구역별 걸레/도구 **분리 사용** (화장실용 ≠ 사무실용)
- 소모품 부족 시 아르민 실장님께 보고

---

## 1. 내부 화장실

### 매일
| 순서 | 항목 | 방법 |
|------|------|------|
| 1 | 변기 내부 | 변기 세정제 + 솔로 꼼꼼히 |
| 2 | 변기 외부/시트 | 세정제 + 걸레로 닦기 |
| 3 | 세면대 | 세정제로 물때 제거 + 물기 닦기 |
| 4 | 거울 | 유리세정제 + 마른 천 |
| 5 | 바닥 | 쓸기 → 물걸레 (배수구 이물질 제거) |
| 6 | 쓰레기통 | 비우기 + 봉투 교체 |
| 7 | 소모품 보충 | 휴지, 핸드타올, 비누 확인 |
| 8 | 문 손잡이 | 세정제로 닦기 |

### 주 1회
- 환풍기/환기팬 외부 닦기
- 타일 줄눈 곰팡이 확인

### 월 1회
- 환풍기 필터 청소
- 배수구 집중 청소

---

## 2. 외부 화장실

### 매일
| 순서 | 항목 | 방법 |
|------|------|------|
| 1 | 변기 내부 | 변기 세정제 + 솔 |
| 2 | 변기 외부/시트 | 세정제 + 걸레 |
| 3 | 세면대 | 세정제 + 물기 닦기 |
| 4 | 거울 | 유리세정제 + 마른 천 |
| 5 | 바닥 | 쓸기 → 물걸레 |
| 6 | 배수구 | 머리카락/이물질 제거 |
| 7 | 쓰레기통 | 비우기 + 봉투 교체 |
| 8 | 소모품 보충 | 휴지, 핸드타올, 비누 |
| 9 | 문 손잡이 | 세정제로 닦기 |

### 주 1회 / 월 1회
- 내부 화장실과 동일

---

## 3. 휴게실

### 매일
| 순서 | 항목 | 방법 |
|------|------|------|
| 1 | 식탁/의자 | 행주로 닦기 |
| 2 | 싱크대/조리대 | 세제 + 수세미 → 물기 제거 |
| 3 | 전자레인지 내/외부 | 젖은 행주로 닦기 (음식물 찌꺼기) |
| 4 | 정수기 외부 | 마른 천으로 닦기 |
| 5 | 바닥 | 쓸기 → 물걸레 |
| 6 | 쓰레기통 | 비우기 + 봉투 교체 |
| 7 | 분리수거 | 정리/배출 |

### 주 1회
- 냉장고 외부 닦기
- 수납장/선반 먼지

### 월 1회
- 냉장고 내부 정리 (유통기한 지난 것 폐기)

---

## 4. 사무실

### 매일
| 순서 | 항목 | 방법 |
|------|------|------|
| 1 | 바닥 | 쓸기 → 물걸레 (or 밀대) |
| 2 | 공용 테이블 | 마른 천으로 닦기 |
| 3 | 쓰레기통 | 비우기 + 봉투 교체 |

### ⚠️ 사무실 주의사항
- **개인 책상 위 서류/장비 절대 이동 금지**
- 개인 책상은 위를 닦지 않음 (본인이 비워둔 경우만)
- 복합기/프린터 외부만 닦기 (버튼 부분 주의)
- 모니터 화면 닦지 않기 (전용 클리너 필요)

### 주 1회
- 창문턱/선반 먼지 제거
- 공용 장비 (복합기, 정수기) 외부 닦기
- 출입문/손잡이

### 월 1회
- 에어컨 필터 확인 (필요 시 보고)
- 구석/책상 뒤 먼지
