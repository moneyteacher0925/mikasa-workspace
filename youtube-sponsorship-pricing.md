# 무농약상토 유튜버 협찬 — 단가 및 실행 계획

> 작성일: 2026년 3월 24일 | 제품: 무농약상토 | 카테고리: 원예/가드닝/텃밭/육아

---

## 1. 조회수 기준별 협찬 단가표 (시장 평균)

| 등급 | 구독자 수 | 평균 조회수 | 보수 방식 | 예상 단가 (1건) |
|------|-----------|-------------|-----------|-----------------|
| 나노 | 1,000~1만 | 500~3,000회 | 제품만 제공 | 0원 (제품 원가만) |
| 마이크로 | 1만~5만 | 3,000~1만회 | 제품 + 광고비 | 25만~50만 원 |
| 중형 | 5만~10만 | 1만~3만회 | 제품 + 광고비 | 50만~100만 원 |
| 대형 | 10만~50만 | 3만~10만회 | 제품 + 광고비 | 100만~300만 원 |

> ※ 원예/가드닝 카테고리는 뷰티·IT 대비 단가가 20~30% 낮은 편
> ※ 위 단가는 브랜디드(전체 영상) 기준, PPL(일부 노출)은 50~70% 수준

---

## 2. 제안 전략: 단계별 접근

### 1단계 (테스트) — 먼저 3~5명

| 항목 | 내용 |
|------|------|
| 타겟 등급 | 나노 + 마이크로 (구독자 5,000~3만) |
| 협찬 수량 | 3~5명 |
| 보수 | 나노는 제품만, 마이크로는 제품 + 25~30만 원 |
| 예상 총 비용 | 75만~150만 원 + 제품비 |
| 목적 | 콘텐츠 반응 테스트 + 전환율 데이터 확보 |
| 기간 | 2~3주 내 영상 업로드 → 1개월 성과 측정 |

### 2단계 (확장) — 성과 기반 스케일업

| 항목 | 내용 |
|------|------|
| 타겟 등급 | 1단계에서 ROAS 좋았던 등급 집중 |
| 협찬 수량 | 10~15명 |
| 보수 | 1단계 데이터 기반 단가 협상 |
| 목적 | 본격 매출 확보 + 검색 노출 장악 |

---

## 3. 광고비 제시 기준 (조회수 기반 CPV)

| 평균 조회수 | CPV (조회당 단가) | 제안 광고비 계산 |
|-------------|-------------------|------------------|
| 3,000회 | 50~80원 | 15만~24만 원 |
| 5,000회 | 50~70원 | 25만~35만 원 |
| 1만회 | 40~60원 | 40만~60만 원 |
| 3만회 | 30~50원 | 90만~150만 원 |

> CPV = 제안 광고비 ÷ 예상 조회수
> 원예 카테고리 적정 CPV: 40~70원

---

## 4. 제품만 제공 vs 광고비 포함 기준

| 조건 | 제품만 제공 | 광고비 포함 |
|------|------------|------------|
| 평균 조회수 | 3,000회 미만 | 3,000회 이상 |
| 구독자 | 1만 미만 | 1만 이상 |
| 채널 성격 | 성장 중인 신규 채널 | 안정적 조회수 나오는 채널 |
| 비고 | 대부분 제품만으로도 수락 | 광고비 없으면 거절 가능성 높음 |

---

## 5. 유튜버별 전용 쿠폰코드 체계

| 유튜버 | 쿠폰코드 예시 | 추적 목적 |
|--------|---------------|-----------|
| A 채널 | SANGTO-A | 유입 경로별 매출 추적 |
| B 채널 | SANGTO-B | 채널별 전환율 비교 |
| C 채널 | SANGTO-C | ROAS 계산 |

> 쿠폰코드는 스마트스토어/쿠팡에 각각 등록
> 할인율: 5~10% 또는 무료배송 권장

---

## 6. 제안서에 포함할 내용

| 항목 | 내용 |
|------|------|
| 회사 소개 | 가든잇(주), 원예용품 전문 기업 |
| 제품 소개 | 무농약상토 — 농약 없는 안전한 흙, 가족·아이 안심 |
| 협찬 내용 | 제품 무료 제공 + 광고비 (해당 시) |
| 콘텐츠 가이드 | 자유 제작, "무농약" 키워드 + 구매링크 필수 포함 |
| 게시 조건 | 수령 후 2~3주 내 업로드, 6개월 유지 |
| 연락처 | 담당자 이름 / 이메일 / 전화번호 |

## 7. 추천 실행 일정

| 주차 | 할 일 |
|------|-------|
| 1주차 | 유튜버 후보 리스트 20명 선정 + 데이터 분석 |
| 2주차 | 1차 타겟 5명 선정 → 제안서 발송 |
| 3주차 | 수락 확인 → 제품 발송 |
| 4~5주차 | 영상 업로드 |
| 6주차 | 성과 측정 (조회수, 쿠폰 사용, 매출) |
| 7주차 | 2단계 확장 여부 결정 |
