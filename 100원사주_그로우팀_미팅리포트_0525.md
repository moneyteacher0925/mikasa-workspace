**일시:** 2026-05-25 (월) 16:00  
**참석:** 에렌, 그로우팀  
**주제:** 100원사주 메타광고 전략 수립

---

## 1. 이벤트 매칭 품질

| 항목 | 내용 |
|------|------|
| 현재 점수 | **4.2 / 10** (낮음) |
| 의미 | 높을수록 전환 데이터 정확, 낮으면 DB 추적 부정확 |
| 개선 방법 | 웹사이트 코드단 이벤트 파라미터 강화 (클로드코딩으로 명령어 보강) |
| 참고 | 모든 웹사이트에 적용되는 건 아님 — 사이트별 확인 필요 |

**전환 이벤트 매핑:**
- 100원 결제 → **장바구니(AddToCart)**
- 유료 결제 → **구매(Purchase)**
- 초반 데이터 부족 시: 장바구니 최적화 먼저 → 데이터 축적 후 구매 전환으로 변경

---

## 2. 캠페인 구조 (ABO → CBO)

| 구분 | 용도 | 설명 |
|------|------|------|
| **ABO** (광고세트예산) | 소재 테스트 | 세트별 예산 배분, 소재 성과 비교에 유리 |
| **CBO** (캠페인예산) | 스케일링 | 위너 소재 발견 후 전환, 자동 예산 최적화 |

**흐름:** ABO로 소재 테스트 → 위너 소재 발견 → CBO로 스케일업

---

## 3. 타겟팅 전략

| 타겟 유형 | 설명 |
|-----------|------|
| **논타겟** | 메타에 완전히 맡김 (알아서 최적화) |
| **어드벤티지 타겟** | 초기 타겟 설정하되 효율 좋은 곳으로 자동 확장 |
| **관심사 타겟** | 사주 관련 관심사 설정 (너무 많이 넣지 말 것) |

**우선 진행:** 논타겟 + 관심사 타겟, 2가지로 동시 테스트

**사주 관련 관심사 탐색 필요** — 사주에 맞는 관심사 카테고리 리서치

---

## 4. 광고 운영 팁

| 항목 | 내용 |
|------|------|
| 종료날짜 | **설정하지 않음** — 6일차에 성과 폭발할 수 있으므로 유연하게 운영 |
| 세트당 소재 | **1~2개** |
| 소재 반응 좋으면 | CPM 자연 하락 (전환, 좋아요, 댓글 등 engagement 반영) |
| 트래픽 광고 | 구매 전환 나오면 보너스인 광고 (기대치 낮게) |
| 기여모델 | 일반(광고 볼 사람 vs 안 볼 사람 비교) / 증대 — 일반으로 시작 |

---

## 5. 광고 소재 가이드

- **AI티 안 나는 소재** — 자연스러운 느낌 필수
- **영상 초반 1~2초 후킹 강하게**
- **광고문구 2줄** 이내로 간결하게
- **사주 이벤트/특별 이벤트** 타겟 소재도 테스트
