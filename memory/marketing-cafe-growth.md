# 네이버 카페 키우기 전략 (엠타트업 4조 강의, 2026-05-22)

## 핵심 원칙
- 업체 티 절대 내지 마라 (비영리 느낌)
- 카페는 쉬어도 불이익 없음 (블로그와 다름)
- 악성키워드/이상한 글만 안 쓰면 카페 안 죽음

## 초반 세팅
- 카페 이름: 3글자, 메인키워드 포함, 세부키워드 4~5개
- 메뉴: 자유게시판 + 질문게시판만 (후기/질문 분리)
- 가입정보: 유입경로 파악 + 검색어 질문
- 등업: 최적화 이후에

## 작업 방식
- 아이디당 50개 투입, 하루 1개씩 글
- IP: 테더링 (고정IP보다 안전)
- 페르소나 통일, 혼자 작성 티 안 나게
- 댓글: "쪽지 보내드렸어요" → 키워드 포함

## 키워드 전략
- 월 최소 3,000개 키워드 배정
- 지역+핵심키워드 전국 깔기 (3개월)
- 블랙키위 활용, 1등 카페 키워드 참고
- 검색량 1이라도 맞으면 깔기

## 유입 채널 (우선순위)
1. 지식인 (매일 50개)
2. 블로그 댓글 + 주인장 섭외
3. 기사 댓글 (매일 10건)
4. 유튜브 댓글 (뜬 영상 + 뜨는 영상)
5. DC 등 커뮤니티
6. 카톡방 (가성비 낮음)

## 유튜브 꿀팁
- 뜬 영상: 댓글 달고 나중에 수정 ("수정됨" 포인트)
- 뜨는 영상: "OOO에서 유명한 거 보고 왔어요"

## 활성화
- 인기멤버를 내 아이디로 확보
- 활동왕 제도
- 글에 바로 피드백 = 핵심
- 경쟁업체 주의

## 최적화 확인
- 사람들 유입 + 단골 10명 = 최적화
- 카페이름 검색 안 나오면 = 죽은 것
