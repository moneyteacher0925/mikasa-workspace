# LiveKlass 운영 가이드 — 진격의 클래스 실전 세팅용

## 📋 실전 세팅 체크리스트

### Phase 1: 사이트 기본 세팅
- [ ] 운영자 계정으로 로그인 확인 (사이트 관리 버튼 보이는지)
- [ ] 사이트 기본 정보 입력 (사이트명, 설명, 로고)
- [ ] 푸터 정보 입력 (사업자 정보 — 온라인 비즈니스 필수)
- [ ] 파비콘 설정 (사이트 관리 → 홈페이지 설정, 권장 16x16 또는 32x32px)
- [ ] 사이트 메뉴 구성
- [ ] 페이지 디자인 (블록 기반 — 텍스트/이미지/배너/게시판 등)
- [ ] 사이트 템플릿 활용 가능 (직무/재테크/어학/피트니스/학원 등 무료 제공)

### Phase 2: 클래스 생성 및 설정
- [ ] 클래스 생성 (클래스명, 썸네일, 소개 작성)
- [ ] 커리큘럼 구성 (강의관리에서 추가)
  - 라이브 수업 (Zoom/Google Meet 연동)
  - 동영상 강의 (직접 업로드 or 유튜브/비메오 링크)
  - 오프라인 모임
  - 디지털상품 (과제, 강의자료, E-book)
- [ ] 섹션 기능으로 강의 그룹 관리
- [ ] 순서대로 진행 여부 설정 (체크 시 순차 학습 강제)
- [ ] 변경사항 저장하기 클릭 (필수!)

### Phase 3: 판매 설정
- [ ] 유료 클래스 가격 설정
- [ ] 프로모션(할인) 적용
- [ ] 옵션 상품 설정 (코칭권 등)
- [ ] 패키지 설정 (여러 클래스 묶음 판매 가능)
- [ ] 모집 기간 & 수강 기간 설정
  - ⚠️ **유료 클래스 수강기간은 최대 3개월** (PG사 계약 조건)
  - 3개월 초과 시 무료 서비스로 간주 → 소개 페이지에 공지 필요
- [ ] 클래스 공개/비공개 설정

### Phase 4: 상세 설정
- [ ] 강사 추가 (클래스별 강사 설정 가능)
- [ ] 수강기간 변경
- [ ] 클래스 카테고리 설정
- [ ] 추가 정보 수집 설정 (소속, 직급 등)
- [ ] 환불 정책 작성 및 게시

### Phase 5: 마케팅 & 검색 등록
- [ ] 네이버 서치어드바이저 등록 (HTML 태그 → 라이브클래스 고객성공팀에 전달)
- [ ] 구글 서치 콘솔 등록 (URL 접두어 방식 → HTML 태그 전달)
- [ ] 구글 애널리틱스 / 페이스북 픽셀 연동
- [ ] 카카오톡 톡상담 설정 (kakao for business 가입 → 프로필 ID `_abcdefg` 형식으로 입력)
  - ⚠️ 오픈채팅방/오픈프로필은 연결 불가
- [ ] 캠페인 관리 (메시지 템플릿, 캠페인 발송)

### Phase 6: 운영 준비
- [ ] 정산 정보 등록 (사업자 계좌, 본인인증, 정산금 1천만원 이상 시 KYC)
- [ ] 사업자 등록 확인
- [ ] 통신판매업 신고 확인 (연매출 8천만원 이상 or 연간 판매 50건 이상)
- [ ] 수강생 안내 준비
  - 회원가입 → 수강신청 흐름 안내
  - VOD: "완료 후 다음" 버튼 클릭 안내
  - 라이브: Zoom/Meet 사전 설치 안내

### Phase 7: 도메인 연결 (유료 플랜 전용)
- [ ] 도메인 구매
- [ ] 채팅 상담으로 도메인 연결 요청
- [ ] DNS 설정값 수령 후 도메인 구매처에서 입력
- [ ] 소셜 로그인 연동 (구글/카카오/페이스북)
- [ ] 연결 완료까지 영업일 2~5일 소요
- [ ] ⚠️ 비즈니스 인증(사업자 등록) 필수

## 💰 수수료 & 정산 구조

### 판매 수수료
| 항목 | 수수료 |
|------|--------|
| **무료 플랜** | 매출의 **10%** + 부가세 별도 |
| **유료 플랜** | 매출의 **5%** + 부가세 별도 |

### PG 수수료 (부가세 별도)
| 결제 수단 | 수수료 |
|-----------|--------|
| 국내 결제 (KRW) | **3.5%** |
| 해외 카드 (USD) | **4~4.5%** + 환전/인출 수수료 |
| 해외 카드 | **4.5%** |
| 페이팔 | **6.5% + $0.3/건** |

### 인앱 결제 수수료 (부가세 별도)
| 플랫폼 | 수수료 |
|--------|--------|
| 애플 앱스토어 | **30%** |
| 구글 플레이스토어 | **30%** |

### 정산 일정
- **정산일**: 매월 익월 10일 (공휴일 시 다음 영업일 +2일 이내)
- **기준**: 유료상품 시작일 기준 (라이브=수업시작일, 오프라인=모임시작일, VOD=결제일)
- **PG사**: 토스페이먼츠
- **송금 시 세금계산서 발행** (수수료에 대해서만)
- 쿠폰 할인 금액은 현재 정산에 미포함 (추후 수수료 부과 가능)

### 정산 유보 사유
1. 정산정보 미등록 또는 불완전
2. 정산정보 명의 불일치
3. 결제완료 대기자 환불/정산 포함 여부 미고지
4. 환불/추가결제 등 변동 발생
5. 본인인증/KYC 미완료
6. 커리큘럼 미확인
7. 클래스 운영 여부 미확인

### 매출 신고
- 운영자가 **직접** 국세청에 부가세 신고 (일반 현금매출=건별매출로 신고)
- 라이브클래스는 매출내역 제공만 함

## 🔄 환불/취소 정책

### 전액 환불
- 수강생 목록 → 체크박스 선택 → "수강취소" → 자동 전액 환불
- 취소된 수강생은 재승인 불가 (재신청 필요)

### 부분 환불
- 매출관리 → 결제 조회 → 주문 상세 → 결제 취소 → "부분" 선택
- ⚠️ **부분 취소는 1회만 가능** — 신중하게!
- "수강 유지" 옵션: 환불하되 수강 권한 유지

### 처리 불가 케이스 (고객성공팀 문의 필요)
- 가상계좌/무통장 입금 결제 건
- 이미 정산 완료/진행 중인 건
- 현금결제의 수강 유지 & 부분 취소

### 정산 완료 후 환불
- 익월 정산금 > 환불금: 익월 정산에서 차감
- 익월 정산금 < 환불금: 환불금 입금 후 처리

## 🏢 사업자 등록 & 통신판매업

### 사업자 등록
- 사업 개시 전 또는 시작일로부터 **20일 이내** 관할 세무서/홈택스에 신청

### 통신판매업 신고 기준
- 간이과세자가 아닌 경우 (연매출 **8,000만원 이상**)
- 직전 연도 판매건수(환불 제외) **50건 이상**
- 간이과세자 또는 50건 미만은 면제 (개정법)
- 그래도 지속 운영 예정이면 미리 신고 권장

### 필요 서류
- 원격평생교육시설/학원 신고도 검토 필요

## 📱 사이트 구조

### 3대 영역
1. **상단 메뉴 영역**: 맞춤형 메뉴 구성, 알림, 프로필, 사이트 관리 버튼
2. **중간 커스텀 영역**: 블록 기반 디자인 (텍스트/이미지/배너/게시판/폼 등)
3. **하단 푸터 영역**: 사업자 필수 정보

### 클래스 상세페이지 구조
- **상단**: 썸네일, 신청/입장 버튼, 클래스 관리(운영자용), 복사/삭제
- **본문 상단**: 클래스 소개
- **본문 하단**: 커리큘럼 (라이브/VOD/오프라인/디지털상품)
- **하단**: 강사 소개, Q&A

## 📢 라이브 수업 운영 팁

### 사전 준비
- Zoom 연동 설정
- 강사에게 Zoom 호스트 권한 부여 (미팅 오픈하는 **강사 1명에게만**)
- 오전/오후 시간 설정 주의

### 수업 당일
- 30분 전 입장: Wi-Fi, 마이크, 스피커, 배터리 체크
- 수강생 전체 음소거
- 준비화면 + 배경음악 공유
- 다시보기 제공 시 녹화 버튼 누르기

### 수강생 알림
- 발송 추천: 수업 전날(24h), 1시간 전, 20분 전
- Zoom 링크 대신 **클래스 링크** 전달 권장
- 마케팅 동의 수강생에게만 발송 가능
- (광고) 표기 필수, 21시 이후 발송 불가

## 📌 기타 참고

- **쿠폰**: 할인 금액은 현재 정산 미포함. 사용 수량/금액 제한 적음. 추후 수수료 부과 가능
- **플랜**: 무료/유료 플랜 존재. 유료 플랜 시 수수료 5%로 절감 + 도메인 연결 가능
- **클래스 복사**: 잘 세팅된 클래스를 그대로 복사해 재활용 가능
- **팝업 기능**: 오픈채팅방 안내 등에 활용
- **결제수단**: 국내 카드, 가상계좌, 계좌이체, 해외카드, 페이팔
- **결제 회사명**: "퓨쳐스콜레"로 표시됨
- **현금영수증**: 수강생이 직접 발급 가능
- **캠페인 관리**: 메시지 템플릿 관리, 캠페인 발송, 부가서비스 결제
