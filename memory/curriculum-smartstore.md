# 진격의 클래스 — 위탁판매 4주 커리큘럼

> **강의명**: 소자본으로 월급 이상, 재고리스크 없는 스마트스토어 판매
> **형태**: 대면 컨설팅 4주 (촬영 → VOD 전환)
> **랜딩**: https://attackclass.liveklass.com/
> **가격**: 149~179만원 (얼리버드/정상가)
> **인원**: 12~15명 한정
> **보장**: 첫 매출까지 무한 피드백
> **플랫폼**: 스마트스토어 집중 + 쿠팡은 추후 추가 퍼널

## 1주차: 시작 — "왜 파는가 + 오늘 바로 시작"

### 대면 컨설팅 (3시간)

**Part 1. 오프닝 & 동기부여 (60분)**

1. **[스토리] 모텔비가 아까워**
   - 여자친구와 있고 싶어도 일 있다고 가봐야 했던 청년
   - 돈이 없으면 이렇게 됩니다

2. **[스토리] 야구선수로 실패한 인생**
   - 집에서 영양가 없는 포지션
   - 온라인 판매를 시작하게 된 이유

3. **[동기부여] 벤츠 타는 2살 많은 누나**
   - 명품을 손쉽게 소비하는 사람들
   - 내 목표: 벤츠, 월 순수익 1,000만원

4. **[자각] 여러분은 왜 돈을 더 벌려고 하나요?**
   - 월급 받으면서 계속 생활하면?
   - 온라인 판매로 얼마까지 벌 건가요?
   - 목표 설정 시간

**Part 2. 판매의 본질 (40분)**

5. **[판매방식] 온라인 셀러 3가지 유형**
   - 제조: 개발자 (아이슈타인, 특허)
   - 사입: 독점권 (라이센스, 돈 많은 사람)
   - 위탁: 나 잘 팔아, 리스크 지기 싫어 (홈쇼핑, 플랫폼)
   - → 우리는 위탁으로 리스크 없이 월 1억까지

6. **[메리트] 판매 6하원칙**
   - 누가, 언제, 어디서, 무엇을, 어떻게, 왜
   - 타겟을 명확하게 설정하는 법

7. **[메리트] USP — 나만의 차별점**
   - 내 상품의 본질은?
   - 권은비의 USP는?
   - 나만의 차별점 만들기

**Part 3. 수익화 구조 + 바로 실행 (80분)**

8. **[실무] 돈 되는 판매 수익화 구조**
   - 소비자가 → 도매가 → 수수료 → 순이익
   - 마진 계산법 (엑셀 템플릿 제공)
   - 완벽하게 배우고 실행 X → 구조만 알고 바로 실행 O

9. **[실무/실습] 스마트스토어 개설 실습**
   - 사업자등록 → 스토어 가입 → 초기 세팅
   - 화면 보면서 같이 진행 (촬영 포인트)

10. **[실무] 도매사이트 가입 + 소싱 기초**
    - 도매꾹/도매매/오너클랜 가입
    - 잘 팔리는 상품 찾는 3가지 방법
    - 위탁 계약 프로세스

11. **[메리트] 이게 가장 중요해! → 아닙니다**
    - 소비자 간파하기
    - 돈 되는 카피라이팅 맛보기

## 2주차: 소비자 이해 — "누구한테 파는가"

### 대면 컨설팅 (3시간)

**Part 1. 복습 + 미션 피드백 (40분)**

1. **[복습] 왜 돈 버는지 다시 짚기**
   - 얼마를 벌어야 자유롭게 살 수 있을까
   - 신라모텔 → 신라호텔

2. **[피드백] 자각미션 피드백**
   - 수강생별 목표 리뷰
   - 잘 쓴 예시 공유

3. **[피드백] "그게 중요한 게 아니야" 미션 피드백**
   - 수강생 제출 분석
   - 핵심을 꿰뚫는 예시 시연

**Part 2. 사고력 트레이닝 (50분)**

4. **[자각] 총량의 저주**
   - 너무 많은 일 = 집중 불가
   - 김치찌개 전문점 vs 분식집
   - 판매 전문가가 되기

5. **[동기부여] 월천 이상 번 수강생 공통점**
   - 동기부여(기름) 없으면 앞으로 못 감

6. **[사고력] 지피지기 백전불태**
   - 상대 꿰뚫기
   - 통찰 4단계: 1인칭 → 전지적 참견 시점
   - 훈수 둘 때 왜 잘 보이는지

7. **[예시] 다이어트약 타겟 실습**
   - 뚱뚱+게으른 타겟 → 맞는 상품
   - 단기간에 빼야 되는 타겟 (결혼식 전) → 맞는 상품
   - 타겟에 따라 바뀌는 상품

**Part 3. 상품 등록 실무 (90분)**

8. **[실무/실습] 키워드 리서치**
   - 네이버 검색광고 키워드 도구 사용법
   - 검색량/경쟁강도 분석
   - 돈 되는 키워드 고르는 기준

9. **[실무/실습] 스마트스토어 상품 등록 A to Z**
   - 카테고리 선택 → 상품명 → 옵션 → 가격
   - 화면 보면서 같이 등록 (촬영 포인트)
   - 상위노출 잘 되는 상품명 공식

10. **[실무] 경쟁 상품 분석법**
    - 상위 노출 상품 뜯어보기
    - 리뷰/가격/배송/상세페이지 비교

11. **[인지] 동기부여 vs 실무, 뭐가 더 중요?**
    - 똑같은 방법 → 월 300 vs 월 3,000 vs 월 3억
    - 여러분 자체를 바꿔야 성과가 나옴

## 3주차: 마케팅 실전 — "어떻게 파는가"

### 대면 컨설팅 (3시간)

**Part 1. 미션 피드백 + 소비자 심리 (60분)**

1. **[피드백] 상품 등록 리뷰**
   - 수강생 등록 상품 화면 공유 + 즉석 피드백
   - 상품명/가격/카테고리 수정 포인트

2. **[학습] 소비자 구매과정 6단계**
   - 현상 → 욕구 → 계획 → 행동 → 대안 → 결과
   - 각 단계를 키워드화해서 찾아보기
   - 나는 어느 단계를 타겟해서 마케팅하고 있나?

3. **[학습] 관여도 — 상품의 2가지 분류**
   - 저관여 (흙, 생필품) vs 고관여 (전자기기, 명품)
   - 관여도에 따라 소비 과정이 완전히 다름
   - 저관여 → 바로 쇼핑검색 구매 / 고관여 → 블로그,카페,유튜브 탐색

4. **[학습] 판매자 판매과정 5단계/8단계**
   - 관여도에 따라 판매자 전략이 달라야 하는 이유
   - 블로그/카페 마케팅이 헛수고가 되는 경우

**Part 2. 상세페이지 + 마케팅 (60분)**

5. **[학습] 유입경로에 따라 달라지는 상세페이지**
   - 쇼핑검색 유입 vs 블로그 유입 vs SNS 유입
   - 각각 다른 상세페이지 전략

6. **[학습] 상품에 따라 달라지는 마케팅**
   - 도마뱀 1,000만원 — 길거리 vs 파충류샵
   - 식품 — 누구나 먹으니까 길거리도 OK
   - 야구용품 — 성수동 핫플에서? 안 팔림
   - 이 개념을 온라인화 하기

7. **[실무] 한눈에 들어오는 썸네일/상세페이지**
   - 첫인상이 전부 (소개팅 5초 법칙)
   - 클릭률 높은 썸네일 공식
   - 캔바/미리캔버스로 만들기

8. **[학습] 소비자 가스라이팅**
   - 의심/반박 포인트를 Q&A로 제거
   - 가격 비싸면 → "성과 안 나오면 환불?" → 사전 제거

**Part 3. 바이럴 마케팅 기초 (60분)**

9. **[실무/실습] 카페/블로그 바이럴 글쓰기**
   - 키워드 배치법
   - 자연스러운 상품 노출 방법
   - 실제로 1개 같이 작성 (촬영 포인트)

10. **[학습] 경쟁자 파이 뺏어오기**
    - 위고비 예시 — 검색량 많은 브랜드 키워드 활용
    - 성분 비슷 + 가격 절반 → 설득력
    - 카페 외부 형식으로 실전 적용

## 4주차: 팔리게 만드는 법 — "돈 버는 사람의 사고방식"

### 대면 컨설팅 (3시간)

**Part 1. 피드백 + 동기부여 (50분)**

1. **[피드백] 상세페이지 + 바이럴 글 리뷰**
   - 수강생별 즉석 피드백
   - 잘한 사례 공유

2. **[동기부여] 되고 싶은 대상들의 삶**
   - 실행력 UP = "어? 이게 되네?"
   - 다이어트 조금 빠지고 "얼굴 이쁘네" 들으면 더 하게 됨
   - 수강생 중 성과 나온 사람 사례 공유

3. **[트레이닝] 똑똑한 놈이 세상을 지배한다**
   - 감독님 새벽 2시 트레이닝 이유 (학부모 마케팅)
   - 당신의 잘못이 아닙니다
   - 좋은 판매자가 되는 법

**Part 2. 고급 판매 심리학 (70분)**

4. **[학습] 제로의 심리 — 필요 없는 소비자한테 파는 법**
   - 당위성: 왜 사야 하는지 이유 만들기
   - 결핍성: 지금 안 사면 없어요
   - 기발성: 이런 건 처음이지?
   - 시급성: 지금 당장!
   - 비교성: 다른 거보다 이게 낫죠?
   - 5가지 기법 실전 적용 실습

5. **[학습] 그들이 원하는 걸 주고 돈 벌기**
   - 라면 에피소드 (내가 원한 건 라면, 엄마가 원한 건 건강)
   - 소비자가 원하는 게 옳은 방법이 아니더라도 → 그걸 줘야 돈이 됨

6. **[학습] 나한테 중요한 건 중요한 게 아니야**
   - 제육볶음 좋아하는 나 vs 떡볶이 좋아하는 여성 타겟
   - 여대 앞 음식점 = 마라탕/떡볶이
   - 내 취향이 아니라 타겟의 취향으로 팔아라

**Part 3. 스케일업 + 마무리 (60분)**

7. **[자각] 시간은 누구에게나 똑같이 흐를까?**
   - 4주 지났는데: 누구는 첫 매출, 누구는 스토어도 안 열음
   - 시간이 아니라 "오늘 뭘 했는가"
   - 목표지향적인 삶

8. **[자각] 친구는 당분간 없다**
   - 방해금지 모드, 전화 무시
   - 가족 밥 OK, 친구 술/여행 = NO
   - 돈 벌겠다면서 놀러 다니는 건 모순

9. **[Q&A] 걱정이 앞서 실행이 안 되는 분들께**
   - 경쟁사 신고하면? → 오히려 좋아, 위협이 됐다는 증거
   - 안 팔리면? → 다음 상품 올리면 됨, 재고 없잖아
   - 수강생 실시간 Q&A

10. **[실무] 스케일업 로드맵**
    - 상품 확장 (10개 → 50개 → 100개)
    - 쿠팡 추가 진출 (다음 퍼널 안내)
    - 광고 세팅 기초 (네이버 쇼핑 검색광고)
    - 리뷰 관리 + CS 대응
    - Value Ladder: VOD → 대면 → 컨설팅 → 대행

11. **[마무리] 수강생 성과 공유 + 다짐**
    - 4주간 성과 발표
    - 첫 매출 무한 피드백 시작 안내
    - 2기 모집 도우미 혜택 (추천인 할인 등)

## Value Ladder 전체 구조

```
[무료] 세미나 (3/16)
  → 리드 확보, 에렌 브랜딩
     ↓
[149~179만원] 대면 4주 + 무한피드백 (1기: 3/23~4/13)
  → 메인 수익, 동시에 VOD 촬영
     ↓
[29~49만원] VOD 단독 판매 (4월~)
  → 자동 수익, 2기부터 사전학습 자료로도 활용
     ↓
[월 50~100만원] 1:1 컨설팅
  → 4주 강의 수강생 중 더 원하는 사람
     ↓
[건당 과금] 마케팅 대행 (진격의 마케팅)
  → 수강생이 성장하면 바이럴/트래픽 대행 클라이언트로 전환
```
