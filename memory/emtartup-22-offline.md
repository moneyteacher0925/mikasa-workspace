## 1주차 1강 (OT + 선택 제어 개론)

- **핵심 주제:** 마인드셋 세팅 + 선택 제어(Choice Control) 개론
- **주요 내용:**
  - **서열 게임 인식**: 자본주의는 서열 게임. 돈·힘·뇌가 서열을 결정. GPT로 동갑 내 자산 순위 확인해볼 것
  - **11개 약속(환경 조성):**
    1. 마이너스 감정 버리기 (서운함, 분노, 배신감, 자존심)
    2. 6주 동안 수강생만 생각하기
    3. 변명 금지 - 주인공 마인드
    4. 완전히 비우고 오기 (고집 버리기)
    5. GPT로 미션하지 않기 (본인 사고력으로 올라간 후 활용)
    6. 사람 정리 (방해되는 관계 정리, 가족과 동업 금지)
    7. 욕구 통제 (식사·수면·취미 등)
    8. 비정상적 공간으로 인식 (초월적 사고)
    9. 미션 성실 수행 + 발표
    10. 결핍 위치 유지 (100억이 기준)
    11. 이 기회가 마지막이라는 절박함
  - **선택 제어 4가지 기법:**
    - **양자택일**: 이지선다 (한쪽만 선택하게) + 더블 바인드 (뭘 선택해도 내 이득)
    - **소거법**: 역피라미드형 (넓은→좁은 범위로 설득) + 무당형 (한 마디로 꿰뚫기)
  - 이지선다 = 주도권 없을 때 / 더블 바인드 = 주도권 있을 때 사용
  - 확신을 강제하는 법: 보기 하나 더 추가하면 비교 심리 작동 → 확신 강화

- **핵심 키워드/개념:** 서열 게임, 선택 제어, 양자택일, 이지선다, 더블 바인드, 소거법, 역피라미드, 무당형, 프레임 탈출
- **실전 적용 포인트:**
  - 상세페이지에 이지선다 금지 → 더블 바인드로 구성
  - 헬스장 1일 vs 1년 가격 구조 = 이지선다 사례
  - 데이트 신청, 직원 채용, 고객 설득 모두 선택 제어 적용 가능

## 1주차 2강 (선택 제어 실전 + 소거법)

- **핵심 주제:** 소거법(역피라미드형 + 무당형) 심화 + 실전 적용
- **주요 내용:**
  - **역피라미드형**: 넓은 범위에서 좁혀가며 YES를 연속으로 받기 → 마지막에 구매 전환
    - "당신은 사람입니까?" → "성인입니까?" → "피부 고민 있으시죠?" 식으로 좁히기
    - 상세페이지 구조 = 역피라미드 (위에서 공감 → 아래로 구매)
    - 한 번에 일관성 있게 작성해야 이탈 방지
  - **무당형**: 타겟을 완전히 꿰뚫어 한 마디로 설득 (최고 난이도)
    - "너 이래서 왔구나" → AI 시대에 가장 유효한 전략
    - 맞추면 소름 → 틀리면 신뢰 완전 상실
    - 미래 트렌드: AI가 개인 맞춤 상세페이지 구현 → 무당형 먼저 선점
  - **미션**: 선택 제어 당한 경험 쓰기 + 매일 뉴스/일상에서 선택 제어 발견하여 단톡방 공유
  - **이득 적용**: 현상 이해 → 남에게 설명 가능 → 이득으로 전환

- **핵심 키워드/개념:** 역피라미드, 무당형, 설득의 좁힘, 끄덕끄덕 3번, 일관성, 커스터마이징
- **실전 적용 포인트:**
  - 쓰레드 물길 = 완전 광역 → 넓은 범위에서 시작해야 함
  - 제품 카테고리 무관하게 역피라미드 3단 구조로 연습
  - 무당형은 통찰력 쌓인 후 도전 (자기 카테고리 충분히 파악 후)

## 1주차 3강 (이커머스 가두리 전략 - 메시지 & 팬덤)

- **핵심 주제:** 고객 DB 활용 + 가두리(팬덤) 전략 + 플랫폼 연결
- **주요 내용:**
  - **4번의 메시지 기회** (주문시/도착시/1주일 뒤/1달 뒤):
    - 처음엔 타겟이 원하는 것만 제공 (맞추기 → 다루기 순서)
    - 카페·유튜브·오픈톡방으로 넘기는 채널로 활용
    - 명분 + 혜택 = 메시지의 핵심 공식
  - **가두리 전략 = 관람차 개념:**
    - 한 번 산 고객을 절대 놓치지 않기 → 팬덤으로 전환
    - 판매 후 팬덤 모집 → 재구매 + 라인업 확장 = 100억의 시작
    - 치킨집 사례: 5년간 7만 팀 놓침 → 카페/유튜브 연결했으면 대박
  - **플랫폼 연결 사례:**
    - 빵집 → 카톡 단톡방(떨이 시간 알림, 갓나온 빵 안내) + 유튜브(빵 만드는 과정) + 카페(빵순이/집빵)
    - 맥주집 → 생맥주 대동여지도 카페 + 라인 관리 정보
  - **박진영 공식:** 타겟 맞추기 → 타겟 다루기 → 팬덤 맞추기 → 팬덤 다루기

- **핵심 키워드/개념:** 4번의 메시지, 가두리 전략, 관람차, 팬덤, 타겟 맞추기/다루기, 제로 마케팅
- **실전 적용 포인트:**
  - 구매자에게 경로 이벤트 메시지 → 카페 가입 유도
  - 사용법/주의사항 안내 → 자연스러운 팬덤 유입
  - 카페/오픈톡은 매력적인 네이밍이 핵심 (연구소 X → 구체적 타겟 O)

## 1주차 4강 (DB 전략 + 부활 편지)

    - 문자 최대 300바이트 (더보기 생기면 이탈)
  - **DB 모으기:**
    - 쿠팡: 안심번호 → 구매확정까지만 제공 → 무조건 저장

    - 탈모 → "암행모사" 카페 (부작용 업체 폭로)

    - 운동용품 → 가성비 타겟 특정 → 보완 제품 클로징
    - 관절 영양제 → 밴드 활용 (어르신 타겟)
  - **개인정보보호법 주의:** 어설프면 형사 처벌 (실제 징역 8개월 판례)

- **핵심 키워드/개념:** DB는 신, 부활 편지, 명분과 혜택, 리콜형/안내형 메시지, 교차 DB, 관람차, 재구매율 80%
- **실전 적용 포인트:**

## 2주차 1강 (플러스 에너지 + 미션 피드백)

- **핵심 주제:** 플러스 에너지 + 프레임 전환 + 1주차 미션 피드백
- **주요 내용:**
  - **플러스 에너지:**
    - 마이너스 에너지 → 플러스로 전환하는 3가지: **장점화, 당위화, 상위화**
    - 손흥민 사례 (임신 협박·3억 사기·위약금 500억+ 위기에서도 유로파리그 우승)
    - "주상체 적응진" = 주어진 상황에 적응하며 진전하기
    - 여러분의 고통은 신문에도 안 날 수준 → 사소한 해프닝으로 인식
  - **1주차 확신 강화:**
    - 15번님 100개 첫 판매 → 포모(FOMO) 발생 → 전체 에너지 상승
    - 플러스 에너지 순환: 받기만 하면 거지 → 반드시 돌려줘야 성장
    - 회초리도 플러스 에너지 (성장에 도움되는 쓴소리)
  - **선택 제어 미션 피드백:**
    - 일상이 전부 선택 제어 (술·담배 시작 계기 = 환경에 의한 선택)
    - 바꿀 수 있는 것은 오직 **내 사고**뿐
    - 주도권 있을 때 → 더블 바인드 / 주도권 없을 때 → 이지선다
    - 역피라미드: 반드시 좁아져야 함 (동일 선상 나열 ≠ 역피라미드)

- **핵심 키워드/개념:** 플러스 에너지, 장점·당위·상위, 프레임 전환, 주상체 적응진, 주도권과 선택 제어
- **실전 적용 포인트:**
  - 날씨도 더블 바인드로 인식 (맑으면 좋고, 비 오면 나를 위한 날씨)
  - 상세페이지 = 더블 바인드만 (이지선다 금지)
  - 역피라미드 연습: 3단 좁힘부터 시작 → 점차 10단, 20단으로 확장

## 2주차 2강 (선택 제어 심화 + 영화 예시)

- **핵심 주제:** 선택 제어 프레임 탈출 + 더블 바인드 vs 이지선다 구분 심화
- **주요 내용:**
  - **영화 예시를 통한 구분:**
    - 마녀2: "얘기하면 죽일 거잖아" = 이지선다 (말하면 죽음/안 하면 생존으로 착각)
    - → "안 해도 죽일 건데" = 더블 바인드 (어차피 죽음, 빨리 말하면 편하게)
    - → 탈출법: 상위 프레임 (내가 가치 있는 존재임을 증명)
  - **프레임 탈출 3가지:** 장점화, 당위화, 상위화
  - **"나한테 할 말 없어"** 분석:
    - 더블 바인드 (할 말 있으면 사과, 없으면 그것도 모르니 사과)
    - 주도권 싸움의 본질 (50:50 관계는 없음)
  - **다이어리 판매 이지선다 예시:**
    - 나쁜 예: "1년짜리로 실패할래? 3개월로 성공할래?" (너무 노골적)
    - 좋은 예: "1년짜리 활용률 17%, 3개월짜리 87%" → 데이터로 자연스럽게 유도
  - **무당형 다이어리:** "네가 실패한 게 네 의지인 줄 알아? 아니야. 이거 써서 다시 와"

- **핵심 키워드/개념:** 프레임 탈출, 상위 프레임, 주도권 게임, 데이터 기반 이지선다
- **실전 적용 포인트:**
  - 고객이 이지선다에 갇혔을 때 → 상위 프레임으로 탈출시키기
  - 역피라미드: 3단법(세 가지로 좁히기)부터 시작
  - 무당형: 권위 + 꿰뚫음이 있어야 반말도 용인됨

## 2주차 3강 (100억 로드맵 + 가두리 카페 미션)

- **핵심 주제:** 100억 목표 설정 + 가두리 카페 네이밍 + 무의식 조종
- **주요 내용:**
  - **100억 로드맵:**
    - 디테일할수록 현실 가능성 상승 (엑시트·현금·이자까지 구체적으로)
    - 관람차 없이 100억 불가능 → 재구매가 핵심
    - 이미 20억+ 모았다 떨어진 사람 = 일을 안 해서 (시간만 지키면 복구 가능)
  - **무의식 조종법:**
    - 쭈꾸미 사례: 헬스장 앞 가게를 무의식적으로 인식 → 검색 → 주문
    - 목표를 "눈에 살짝 보이는 곳"에 배치 (대놓고 X, 슬쩍 보이게 O)
    - 목표 차/집/금액 등을 무의식적으로 볼 수 있는 위치에 배치
  - **가두리 카페 미션 피드백:**
    - 안 되는 예: "창문형 에어컨 설치의 모든 것" (1회성, 활동 이유 없음)
    - 좋은 접근: 타겟 욕구로 봐야 함 → 창문형 에어컨 구매자 = 원룸 거주자 → "원룸 생존기"
    - 수면 영양제 → "3교대" 커뮤니티 / 수면제 단약 모임
    - 뷰티 → "홈카리나" (집에서 예뻐지기) / 외모정병 세분화
    - 키 성장 → "약수키" (약X 수술X로 키 키우는 부모 모임)
    - 과민대장 카페도 OK (후킹글 하나가 핵심)
  - **카페 활성화의 어려움:** 가입까지는 쉬움 → 자체 활동이 핵심 난관
    - 통제된 카페가 오래 감 (자유 글 없이 후기+동굴만 = 엠타 카페 방식)

- **핵심 키워드/개념:** 100억 로드맵, 무의식 조종, 가두리 카페, 네이밍, 세분화, 후킹글
- **실전 적용 포인트:**
  - 카페명 = 타겟의 욕구에서 출발 (제품명 X)
  - 기존 대형 카페와 정면 승부 X → 세분화로 틈새 공략
  - 메시지 → 카페 유입 → 후킹글로 가입 유도 (연결 매끄러워야)

## 2주차 4강 (레이어·궤도·패턴)

- **핵심 주제:** 패턴 바꾸기의 3단 구조 (레이어/궤도/패턴) + 미션 체크
- **주요 내용:**
  - **레이어 (축적된 행동 층):**
    - 태어나기 전(유전자 5~7대)부터 중첩된 행동 레이어
    - 조카가 매형 노래 스타일 그대로 (세 살 때 매형 사망했는데도)
    - 엠타 기간에 잠깐 바꿔봤자 기존 레이어가 더 강함 → 다시 돌아감
    - **바꾸는 법:** 행동을 바꾸는 게 아니라 **정체성(동기)을 바꿔야** 함
    - 술 끊기: "나 술 끊을 거야" (실패) vs "나 원래 술 안 좋아했어" (성공)
    - 회피형: "회피형 아니야"가 아니라 "막상 해보니 신나던데" 식으로
  - **궤도 (행동의 방향):**
    - 현재 행동을 보면 과거와 미래가 보임
    - 일진 → 아부의 달인 → 평생 왕 찾는 궤도
    - 미션 하나 대충 = 미래가 별볼일 없음 (궤도로 예측 가능)
  - **패턴 (고착화된 자극-반응):**
    - 파블로프의 개 = 외부 자극 → 고정 반응
    - 칫솔 사례: 세게 닦는 패턴 7~8년째 동일 (새로 사도 같은 제품)
    - 잘될 때 행동 vs 위기 때 행동 적어보기 → 패턴 투명하게 보기
  - **습관/패턴/루틴으로 퉁치지 말고 3개를 분리해서 레이어부터 바꿀 것**

- **핵심 키워드/개념:** 레이어, 궤도, 패턴, 정체성 전환, 고무줄 개념, 파블로프
- **실전 적용 포인트:**
  - 패턴을 바꾸려면 정체성부터 (행동 변화 X → 인식 변화 O)
  - 자신의 "잘될 때/위기 때" 행동 패턴 기록 → 수익 예측 가능
  - 미션·제약 대충하는 패턴 = 돈 못 버는 궤도

## 2주차 5강 (사고력 + CV 본질 꿰뚫기 + 부활편지 피드백)

- **핵심 주제:** 사고력(CV - 본질 꿰뚫기) + 부활 편지 피드백 종합
- **주요 내용:**
  - **사고력 = 현상을 꿰뚫고 프레임을 자유롭게 벗어나는 능력**
  - **CV(Core Value) = 핵심, 본질, 진짜, 속내:**
    - 생선의 뼈 = 본질 → 살(표면)만 보지 말고 골조를 파악
    - 철학은 "왜"에서 시작 → "진짜 왜지?"까지 한 단계 더 깊이
    - 왜 살지 / 왜 힘들지 / 왜 유명한지 → 본질 탐구
  - **본질 사례:**
    - 유재석: 본질 = 압도적 진행력 (300명 3시간 완전 장악) + 이미지 유지
    - 카리나: 본질 = 얼굴 (콧볼만 넓어져도 끝남, 노래·춤은 부차적)
    - 인간의 불행: 대부분 일어나지 않은 일 걱정 → 본질적으로 안 힘든 경우가 대부분
  - **부활 편지 피드백 종합:**
    - 스팸 ≠ 편지: "○○회사입니다, 1주년입니다" = 스팸 → 친구에게 쓰듯 편지체로
    - 좋은 사례: 64번님 (26분 한정, "덕분에 원룸→ 더 나은 환경 이사", "선물" 워딩)
    - 탄핵 굿즈 판매자: 정치 색깔 = 뚜렷한 타겟 → 시리즈별 이어가기 + 같은 편 프레임 활용
    - 온수매트: 보관법 안내(명분) + 파우치 반값(혜택) = 무난한 정석
    - 금지: 카페+오픈톡 동시 유도(욕심 과다), 매뉴얼형 스팸 메시지
  - **사고 테스트 미션 발표:**
    - "사람들은 왜 자동차를 타는가" → 표면: 우월감 / 깊은 답: 타겟에 따라 다름 (정치인 = 검소해 보이기 위해)
    - 의도를 꿰뚫는 연습이 사고력의 핵심

- **핵심 키워드/개념:** CV(본질), 왜의 탐구, 편지체 메시지, 명분+혜택 정석, 사고력 트레이닝
- **실전 적용 포인트:**
  - 부활 편지는 진짜 편지처럼 (스토리텔링 + 진정성)
  - 명분: 제로 마케팅(경고/문제점) + 혜택: 재구매 할인/정보 제공
  - 카페 유도 시 한 번에 하나만 (욕심 부리면 전부 실패)
  - 모든 현상에 "진짜 왜지?" 질문하는 습관 = 사고력 향상의 시작

## 핵심 프레임워크 요약

### 🎯 선택 제어 4기법
| 기법 | 방식 | 사용 시점 |
|------|------|-----------|
| 이지선다 | A or B (한쪽만 선택 가능하게) | 주도권 없을 때 |
| 더블 바인드 | A or B (둘 다 내 이득) | 주도권 있을 때 |
| 역피라미드 | 넓은→좁은 범위로 YES 연속 | 상세페이지, 대면 설득 |
| 무당형 | 한 마디로 꿰뚫기 | 통찰력 축적 후 |

### 📊 가두리(관람차) 전략
1. **판매** → 2. **4번 메시지** (명분+혜택) → 3. **팬덤 유입** (카페/톡방) → 4. **재구매+라인업 확장** → 5. **100억**

### 🔄 프레임 전환 3가지
- **장점화:** 마이너스를 장점으로 해석
- **당위화:** 당연한 과정으로 수용
- **상위화:** 더 높은 프레임으로 올라가기

### 🧠 패턴 변화 3단계
1. **레이어** (유전자~현재 축적) → 정체성을 바꿔야 깨짐
2. **궤도** (행동의 방향) → 현재 행동 = 미래 예측
3. **패턴** (자극-반응 고착) → 잘될 때/위기 때 행동 기록으로 파악

### 💡 부활 편지 공식
**명분** (왜 연락했는지) + **혜택** (받는 사람 이득) = 거부 반응 없는 재활성화
- 스팸 X → 편지체 O
- 내 이득 먼저 X → 타겟 이득 먼저 O
- 300바이트 이내 (더보기 방지)
