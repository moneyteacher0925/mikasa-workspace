## 💡 핵심 인사이트

### 1. "결정사" vs "결혼정보회사" 전략 차이
- **결혼정보회사** (14,290) → 인기글 노출 O → **블로그 SEO** 유효
- **결정사** (12,540) → 인기글 노출 X → **검색광고** 중심 전략 필요
- 결정사 하위 키워드(비용/후기/가격/순위 등) 전부 인기글 X

### 2. 검색량 대비 SEO 기회
- **결정사 비용**(1,630)과 **결정사 후기**(1,530)는 검색량 높지만 인기글 X → 광고만 가능
- **결혼정보회사 추천**(770)은 인기글 O → 블로그로 진입 가능한 고가치 키워드

### 3. 지역 키워드
- **부산**이 지역별 최대 (결정사 810 + 결혼정보회사 1,020 = 1,830)
- 강남 결혼정보회사(420)가 서울 결혼정보회사(120)보다 3.5배 높음 → 강남 타겟팅 효과적

### 4. 모바일 집중
- 전체 평균 모바일 비중 **약 72%**
- 맞선(87.5%), 결혼상담소(86.7%), 부산 결정사(82.7%) 등 일부는 85% 이상
- **모바일 최적화 필수**

### 5. 경쟁도
- **거의 모든 키워드가 "높음"** → 검색광고 비용 높을 것으로 예상
- 유일하게 **결혼정보회사 채용**(60건)만 경쟁도 "중간"
