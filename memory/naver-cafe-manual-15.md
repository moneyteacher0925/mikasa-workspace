# 네이버 카페 바이럴 작업 매뉴얼 - 전체 정리

> **원본 출처:** [N 카페 작업 매뉴얼 (Notion)](https://west-havarti-19d.notion.site/N-355a2e3e108480a38f7fdeaac7ae8398)
> **정리일:** 2026-05-11

---

## 📌 매뉴얼 목적 (Why?)

N사(네이버) 카페를 활용하여 자사 상품을 노출시키고, **판매 플랫폼으로 유입 → 구매전환**을 달성하기 위한 마케팅 작업 매뉴얼.

**핵심 원칙:**
- 모든 작업은 **광고로 인식되지 않도록** 일반회원처럼 자연스러운 전개 필수
- 오직 **크롬 브라우저**만 사용 (엣지 X, 파이어폭스 X, 웨일 X)
- **윈도우 운영체제만** 가능 (Mac/iOS 불가)
- PC와 연결 가능한 **개통된 안드로이드(갤럭시) 기기** 필요
- AI를 사용한 단순업무이므로 **작업 간 실수 절대 불가**

## 🔄 전체 작업 플로우 (순서)

```
1. 키워드 시트 확인 & 본인 이름 기입 (중복 방지)
2. 프롬프트 세팅 (제목 + 메인 키워드 수정)
3. 클로드 AI로 본문글 작성
4. 이미지 수집 & 재가공 (캡처→편집→각도/색상/좌우반전)
5. CCleaner로 트래픽 청소 (Windows + 애플리케이션 각 2회)
6. 테더링으로 IP 변경
7. 크롬 시크릿 모드 → 네이버 로그인 (메모장에서 복붙)
8. 카페 검색 접속 → 글쓰기 → 게시판 선택 → 본문+이미지 업로드
9. 등록 후 로그아웃
10. 다시 5번부터 반복 (다른 아이디로 전환)
11. 본문글 완료 후 댓글 작업 (1번째→2,3번째(AI)→나머지)
12. 급여시트 작성
```

## 📋 섹션별 핵심 내용 요약

### 1. 작업 전 필독사항 ⭐⭐⭐

- **텍스트 복붙 규칙:** 모든 텍스트(아이디/비번, AI 원고, 댓글 등)는 반드시 **윈도우 메모장**에 복사→붙여넣기 후 사용
  - 구글시트/엑셀에서 직접 복붙 금지 → 아이디 사용불가 원인
  - 스티커 메모 금지, Mac OS 금지
- **IP 변경 필수:** 로그인/로그아웃 시마다 IP 변경 (매뉴얼 미준수 시 아이디 사용불가)
- **노트북 사용 금지**, 하나의 데스크탑 PC만 사용
- **아이디 배정:** 작성자용 10개 + 댓글용 10개 (본인 이름으로 배정된 것만 사용)
- **업로드 주기:** 아이디 1개당 하루 최대 2개 포스팅 (A카페 1개 + B카페 1개) → 10개 아이디로 총 20개 가능
- **연속작업 금지:** 한 동작이 끝난 후 다음 동작 진행 (이중 작업 X)

### 2. 테더링 IP 변경방법

- **안드로이드 USB 테더링**으로만 IP 변경 (다른 방법 절대 금지)
- 갤럭시 S24 이상 최신기종은 테더링 불가할 수 있음
- **IP 변경 전 항상 기존 랜선/와이파이 OFF** 필수
- Wi-Fi 아이콘이 안 보일 경우: 검색 → "네트워크 연결 보기" → 기존 랜선 "사용 안함" 처리
- USB 테더링만 켜진 상태로 작업

### 3. 작업할 키워드 확인방법

- "네이버 카페 키워드" 시트 → "작업할 키워드" 시트 확인
- 기입할 항목 5가지: **작성할 카페, 제품 종류, 업로드 날짜, 작가이름, 작업물 링크**
- **반드시 작업 전 "작성할 탭"에 본인 이름 미리 기입** (중복 작업 방지)
- 급여시트의 "업로드 날짜시간" 시트에도 동일하게 기재
- 중복 키워드 발견 시 → 담당자에게 말하고 다른 아이디로 작업
- **특정업체/상호명 키워드:** 제목이나 댓글에 너무 부정적 작성 금지 → 신고/제재 위험 증가

### 4. 본문글 프롬프트 세팅방법

- "네이버 카페 키워드" 시트 하단에 제품별 프롬프트 시트가 분류되어 있음
- **수정할 항목 2가지만:** "제목"과 "메인 키워드"
  - "제목 :" 뒷부분만 수정 (← "제목 :" 자체는 절대 삭제 금지)
  - 메인 키워드: 빨간박스 부분만 교체 (띄어쓰기 동일하게)
- 다른 작업자도 같이 쓰는 프롬프트이므로 **위 2가지 외 다른 부분 수정 절대 금지**

### 5. 클로드 AI를 이용한 본문글 작성법

- **Claude** (정보형 기반 AI) 사용, 담당자에게 계정 확인
- 로그인: "Google로 계속하기"로만 로그인 (이메일 로그인 X)
- **모델: Claude Sonnet 4** 선택 필수
- **새 채팅방 프롬프트 먼저 입력** → 그 후 본문글 프롬프트 입력
  - 새 채팅방 프롬프트 없이 바로 작업하면 오류 잦음
- 채팅방 이름: "이름 날짜" 형식으로 변경 (예: "홍길동 20250517")
- 채팅방 데이터 찰 때까지 하나의 채팅방에서 계속 작업
- 다른 작업자도 같은 계정 사용 → 반드시 이름+날짜 표기

### 6. 이미지 작업 방법

- 타사 제품 사진 사용 시: **오래된 스마트스토어/쿠팡 리뷰, 상위노출 안 된 오래된 블로그/카페 글**에서 가져오기
- **한 계정(인물)의 이미지를 통일**하여 사용 (일관성 유지)
- 본문 내용과 싱크 맞는 이미지를 적재적소에 배치
- 본문글 1개당 필요 이미지: 화자(사람) 이미지 + 상위노출 포스팅보다 1장 더
- **절대 원본 다운로드 금지** → 반드시 **캡처(스크린샷)**로만 가져오기 (파일 고유 데이터 제거)
  - 윈도우 Print 키 또는 **Snipaste(강추)** 사용
- **썸네일(대표) 이미지:** 화자(인물) 이미지를 최상단 1~2개 배치 (클릭/후킹/어그로 효과)
- **유사 이미지 재사용 금지:** 유사문서 감지 → 포스팅 노출 차단 + 아이디 위험

### 7. 이미지 재가공 방법

- **중복/유사 사진 방지**를 위해 반드시 이미지 재가공 필수
- 윈도우 기본 사진첩 프로그램으로 가능
- **필수 3가지 작업:**
  1. **각도 변경:** 좌우 약간만 조정
  2. **좌우 반전:** 우측 하단 화살표
  3. **색상 조정:** 밝기 -20, 노출 -20, 채도 -13, 따뜻함 -25, 색조 14 (대략적 수치)
- 추가: 펜 기능으로 눈 가리기 등 (필수는 아님, 상황에 맞게)
- 본인이 쓰는 이미지 편집 툴/포토샵 사용 가능 (담당자에게 포토샵 계정 요청 가능)

### 8. 작업 간의 IP 변경 ⭐⭐⭐

- **아이디 전환 시마다 IP 변경 필수**
- ⭐ **작업순서: CCleaner 트래픽 제거 → IP 변경**
- IP 변경 전 항상 랜선/와이파이 OFF
- **같은 아이디**로 A카페 → B카페 작업 시에는 IP 변경 불필요
- **다른 아이디**로 전환할 때만 IP 변경

### 9. CCleaner를 사용한 트래픽 청소작업 ⭐⭐⭐

- **목적:** PC에 쌓인 캐시/히스토리/쿠키 등 잡다한 트래픽 제거 → 미제거 시 아이디 전부 날아감
- ⭐ **작업순서: CCleaner 트래픽 제거 → IP 변경**
- **CCleaner 실행 전:** 크롬 완전 종료 (백그라운드 포함) → 안 끄면 크롬 트래픽 못 지움
- **Windows 탭:** 맞춤형 정리 → Windows → 클리너 실행 × **2회**
- **애플리케이션 탭:** Google Chrome에서 "저장된 비밀번호"와 "저장된 양식정보"만 **제외** → 클리너 실행 × **2회**
- 질문창은 모두 "예(Y)" / "계속" 클릭

### 10. 시크릿 모드를 통한 네이버 로그인 ⭐⭐⭐

- **반드시 크롬 시크릿 모드**(Ctrl+Shift+N)에서만 로그인
- 기존 크롬창과 시크릿 모드(회색) 혼동 주의
- 메모장에 저장한 아이디/비번을 **복사→붙여넣기**로만 입력
- **IP 보안 OFF** 상태로 로그인
- "새로운 환경에서 로그인" 메일 → "예" 클릭 (매번)

### 11. 본문글 업로드 방법

- **카페 접속 방법:** 링크 직접 붙여넣기 금지 → **네이버에서 카페명 직접 검색**하여 접속
- 제품군에 맞는 게시판 선택 (모르면 담당자에게 문의)
- 메모장 원고 → Ctrl+A → Ctrl+C → 카페 글쓰기에 Ctrl+V
- 등록 후 반드시 시크릿 모드 로그아웃 & 종료
- **제목에 "걱정" 단어 사용 금지** (부정적 어감 + 업체 신고 위험)
- **연속 업로드 최소 3분 간격** (어뷰징 판별 방지)
- **TIP:** 본문 붙여넣기 후 이미지 찾기를 맞춰 진행하면 자연스러운 시간 간격 확보

### 12. 본문글 댓글 작업방법

- **본문글 먼저 작성 후 댓글 작성** (순서 중요)
- 아이디 전환 시 CCleaner → IP 변경 동일 적용
- 댓글 아이디 닉네임은 자유 변경 가능 (같은 카페 내 중복 금지)
- **댓글 작성 순서:**
  1. **1번째 댓글:** "저도 ㅇㅇ하는데 (메인 키워드) 고민 중이에요 ㅠㅠ" 같은 자연스러운 공감 댓글
  2. **2~3번째 댓글 (핵심!):** 구매 전환이 일어나는 댓글 → **클로드 AI "카페 댓글 프롬프트"** 사용하여 작성
  3. **나머지 댓글:** 메인 키워드 포함하여 가볍게 작성
- **대댓글:** 작성글 아이디로만 진행 (효율성), 댓글용 아이디로 먼저 순차적으로 달고 → 한꺼번에 대댓글
- 클로드 AI 프로젝트 → "카페 댓글 프롬프트" 접속 → "너의 임무는 뭔지 설명해줘" 먼저 입력 → 시트의 댓글 복붙용 데이터 입력

### 13. 급여시트 사용법

- 담당자에게 "ㅇㅇㅇ님 급여시트" 요청
- 수정 항목: **작업 파일 갯수** + **인적사항(성함, 계좌번호, 주민번호)** (3.3% 프리랜서 소득공제)
- 일일 마감마다 작업완료 갯수 기입
- 월 마감 후 시트 복사 → 시트명 & 날짜/월일 수정
- **급여 지급일:** 매달 15일 (1~14일분), 매달 1일 (15~31일분)

### 14. 기타 안내사항

- 시트 내 잘못 기입된 내용 발견 시 → 즉시 담당자에게 연락
- 클로드(제미나이) AI 구글 보안 인증 요청 시 → 담당자에게 연락
- **프로필 사진 변경:** 자연스러운 작업을 위해 사용 아이디의 **50% 정도** 프로필 사진 변경 권장

### 15. 댓글 내 이미지 작업방법

- **구글 제미나이 AI**로 비포/애프터 이미지 생성 (담당자에게 회사 계정 요청)
- 반드시 **"사고 모드"**로 변경 후 작업
- **레퍼런스 이미지를 먼저 제공** → 부분 변경 요청 (처음부터 생성 X)
- **워터마크(우측 하단)** 반드시 제외하고 부분 캡처
- 작업 안 될 때 → "새 채팅" 열어서 다시 시도
- **속도 향상 TIP:** 크롬 탭 여러 개 → 채팅방 1개당 사진 1개씩 동시 요청
- 10개 정도 만들어 돌려가며 사용

## 💡 에렌을 위한 인사이트

### 운영 구조 분석
- **작업자 관리:** 아이디 시트 + 키워드 시트 + 급여시트로 체계적 관리
- **품질 관리:** 프롬프트 표준화 + AI(Claude/Gemini) 활용으로 일관된 콘텐츠 품질
- **리스크 관리:** IP 변경, CCleaner 트래픽 청소, 시크릿 모드, 메모장 복붙 등 다층 보안 체계

### 핵심 노하우
1. **구매전환 핵심은 2~3번째 댓글** → 여기에 AI 프롬프트를 집중 투입
2. **자연스러움이 생명:** 광고 티 나면 신고/제재 → 일반회원처럼 행동
3. **이미지 전략:** 화자(인물) 이미지를 썸네일 최상단 배치 = 클릭률(어그로) 극대화
4. **유사문서 감지 회피:** 이미지 재가공 3종(각도/반전/색상) + 캡처만 사용
5. **테더링 IP 변경:** 같은 아이디 내 카페 이동은 IP 불변, 아이디 전환 시만 변경 → 효율성
6. **제미나이 이미지 생성:** 레퍼런스 기반 부분 변경이 핵심 (맨땅에서 생성 X)
7. **병렬 작업 TIP:** 본문 붙여넣기 후 이미지 찾기를 동시 진행 → 3분 간격 자연 확보

### 대행사 운영 시 참고
- 작업자 온보딩 매뉴얼로 매우 체계적 → 이 수준의 SOP가 필요
- 프롬프트 관리가 구글 시트 기반 → 중앙 집중 관리 용이
- 급여 체계: 작업 파일 갯수 기반 + 3.3% 프리랜서 → 성과급 구조
- 클로드 AI + 제미나이 AI 이중 활용: 텍스트는 Claude, 이미지는 Gemini
- **확장 시 병목:** 안드로이드 기기 확보, 아이디 수급, 카페 제재 리스크 관리
