# 엠타트업 21기 추가 비대면 강의 요약

## 첫번째 비대면 (251120) — 1시간 37분

### 핵심 주제: 욕구의 형상화 + 디테일 + 미래 대비

---

### 1. 욕구의 형상화 (핵심 신규 개념)

**개념**: 제품(R)을 먼저 생각하지 말고, 타겟의 욕구만 깊이 조각(형상화)하라.

- 알(R)은 나중 문제. 욕구만 구체적으로 그리면 결제는 따라온다
- "영수 화법" — 나는솔로 28기 영수 캐릭터에서 착안
  - 상대가 듣고 싶은 말만 정확히 해주는 기술
  - 영수는 책임 없이 했기에 사기꾼 → 우리는 책임까지 지면 "무서운 무기"
- **공식**: 타겟의 욕구를 이상적으로 형상화 → 현실적으로 내가 책임질 수 있는 범위로 가져오기 → 둘을 만나게 하는 것

**실전 예시 3가지**:

1. **여유증(남성 가슴)**: 
   - 타겟 = 첫 성관계 직전 or 여친과 수영장 가야하는 남성 (애정욕구)
   - "이 정도는 여유증도 아닌데? 운동+마사지로 3개월이면 없어지는데요?" 
   - 의사의 전문성을 뺀 영역으로 가져와서 구현

2. **소추(외소증)**:
   - 타겟이 듣고 싶은 말 = "수술 없이 키워줄게" + "이 정도 크기가 수술하기 가장 좋은데요?"
   - 핵심: 작은 사람들은 발기 후보다 **평상시 크기**에 스트레스 → 이 욕구를 정확히 짚어야

3. **탈모**:
   - 깊이 들어가야 — 단순 "탈모샴푸" 아님
   - 타겟이 가장 듣고 싶은 말: "탈모는 이제 불치병이 아닙니다"
   - 줄기세포 연구 논문 + 두피 연결고리 → 비전문가 영역으로 현실화

**핵심 원칙**:
- 알(제품)은 뭐든 된다 — 전자책, 컨설팅, 위탁제품, 서비스 다 가능
- 욕구를 얕게 보면 100개밖에 못 팔아 → 깊이 들어가면 창조자가 됨
- "베껴서 돈 버는 심리"는 다음 카테고리에서도 반복됨 → 창조자가 돼야

---

### 2. 관찰 (자관인간: 자각-관찰-인지)

- 나는솔로처럼 전체적 관점에서 보면 관찰 쉬움 → 내가 그 안에 들어가면 "나"라는 변수 때문에 관찰 어려움
- **일회관지(一回貫之)**: 연애, 사업, 가족 — 한 사람의 패턴은 모든 영역에서 동일
  - 직업을 통해 연애 패턴까지 꿰뚫기 가능
  - 얄팍한 정보비대칭으로 이득 보는 직업(핸드폰, 중고차, 부동산) → 연애에서도 얄팍한 거짓말

---

### 3. 미션 피드백

**타겟 프레임 조종 미션 (1주차)**:
- 필터 = 고통을 성장 버섯으로 바꾸는 것 (슈퍼마리오 버섯 비유)
- 온라인이 오프라인보다 낫다는 필터 → 뛰어난 사고
- 롱패딩 vs 크롭패딩: 패션은 판단력 없는 소비자를 이용 → 없는 걸 만들어서 팔기

**욕구 미션**:
- 연애 × 정신병 = 엄청난 시장 (어플, 결정사)
- 결정사 가는 사람들이 결혼 못 하는 이유: 자기보다 높은 급이 자기 급이라 착각
- "미의 저주" — 아름다운 사람이 행복할 확률 매우 낮음 (무조건적 사랑의 앵커링)
- "고귀한 찐따" — 도전하지 않고 상대가 알아서 다가오길 기다리는 심리

---

### 4. 전쟁 / 경제위기 / 초인플레이션 대비

**전쟁**: 
- 사전 준비 (여권, 비자, 가족 설득) → 즉각 반응 가능하게
- 본인의 불안이 정신병일 수도 있다는 자각까지 포함 (더블 바인드)

**경제위기/초인플레이션**:
- 원화 망하는 건 확정 → 달러 기반 투자
- 멘탈 관리가 경제위기 대응력
- 직원 시스템 갖춰야 — 혼자 7~8천 한계, 3~8명이면 월 3~5억 가능
- 직원에게 착한 척 하지 말 것 — "돈 벌러 온 거" 마인드 (김솔지 대표 인용)

---

### 5. 프로모션 디테일

- 50% 할인 vs 1+1 → 완전히 다른 용도
  - 1+1: 소모품, 쟁여놓을 수 있는 것
  - 50%: 한 번 사고 끝나는 것
- 고관여 상품(탈모 줄기세포 샴푸): 할인 프로모션 X → 프리미엄이 오히려 이벤트
- "모른다고 생각하는 게 더 도움" — 보이스피싱 사례

---

### 6. 김솔지 대표 언급
- 욕구 형상화를 완벽하게 하는 인물
- 본인이 느꼈던 욕구로만 접근 (남의 것 X)
- 틱톡으로 하루 100개+, 1억+ 매출
- 직원 관리 노하우 뛰어남
- 10인 미만 사업장 운영자에게 추천

---

### 에렌에게 적용 포인트

1. **식물미스트 상세페이지**: 식집사의 욕구를 형상화하라 — "비료를 주는 의무감" 아닌 "내 식물을 케어하는 힐링 행위"를 듣고 싶은 말로 구현
2. **분갈이세트 상세페이지**: "처음이라 불안한" 초보 식집사가 가장 듣고 싶은 말 = "이 세트 하나면 실패할 수가 없는데요?"
3. **무료세미나 원고**: 영수 화법 적용 — 타겟(부업 관심자)이 가장 듣고 싶은 말을 정확히
4. **강의 컨설팅**: 수강생의 욕구를 형상화 → "첫매출 무한피드백"이 바로 이것
5. **프로모션 구조**: 진격의 클래스는 고관여 → 할인 X, 가치더하기(1:1 컨설팅 추가) = 정답

## 두번째 비대면 (251203) — 1시간 49분

### 핵심 주제: 배제 + 성장의 한계 + 프레임 분리 심화 + 적의적 프레임

---

### 1. 왜 우리는 "딱 이만큼"만 성장할까

- **배제(排除)**: 성공한 사람들의 공통점 = 이성, 호르몬, 부차적인 기회를 모두 배제
- 유재석: MC 하나만 집중, 빌딩투자/술자리/타직업 전부 배제 → 수십년간 정상 유지
- 오타니: 야구 하나만, 이성 배제, 가족도 배제에 가까움
- **우리의 문제**: 이커머스하면서 이것저것 다 하려 함 → "직업이 바뀌는 수준"의 카테고리 변경을 가볍게 생각
- **제이커브 성장**: 실험→고통→성장→폭발적 성장. 대부분 고통 구간에서 포기 → "빨간선에서 끊김"

### 2. 성장을 막는 요소들
- **자존심**: 네임드가 되면 쉬운 시도도 머뭇거림 → "내가 이 정도인데"
- **미련(매몰비용)**: 이전 제품/플랫폼에 대한 미련이 새 시도를 막음
- **명예**: 유튜브 얼굴 까기 못하는 이유 = 과거 지인이 볼까봐 (망상에 가까움)
- **8~12개월 정체**: 이 기간동안 시도해도 안 되면 → "내 판단의 한계"를 자각해야 함 → 남의 말대로 해야 하는 시점

### 3. 프레임 분리 미션 피드백 심화
- **가치 전달이 없는 프레임 분리는 의미 없음**: "나는 MD 출신이다" = 프레임 분리 아님 → "나는 MD 출신이라 지마켓에 꽂아줄 수 있다" = 프레임 분리
- **빡빡이 10년 → "잔디 인형"**: M타트업에서 머리 기르는 길이 = 수익 척도 → 스토리텔링 완성
- 에미넴 = "화이트 초콜릿" → 단어 하나로 스토리텔링 끝

### 4. 유산균 시장 프레임 분리 실전
- 유산균의 구매 CV 순서: 결핍(변비) → 다이어트 → 장활동 → 피부 → 면역력
- **새 시장 창출**: 유산균 + 부임(정자질 개선) → 프레임 분리 자체가 새 시장 진입
- **스트레스/정신건강 + 유산균**: 장-뇌 연결고리 → "자살 방지 보조제"까지 확장 가능
- 핵심: 제품의 성분이 아닌 **질병 시장에서 프레임 분리**

### 5. 적의적 프레임 핵심
- **"당신 잘못이 아닙니다"** = 마법 카피. 타겟을 자각시키면 이득 X, 프레임 안에 가두고 편들어야 이득
- 상대를 성장시키려면 → 자각시켜라
- 상대에게 돈 쓰게 하려면 → 절대 자각시키지 마라, 편들어라
- **가까운 사람을 적으로 만드는 전략**: 여성타겟에게 남편을 적으로 / 남성타겟에게 아내를 적으로 → 효과적

### 6. 자산/투자
- 원화 가치 하락 확정 → 달러 전환 강조 (엠군 본인 170만 달러 MMF)
- 하급지 부동산은 빠지기 어려움 → 상급지 갈 거면 유동성 높은 자산(달러)이 낫다
- 직원에게 지분% 절대 주지 말 것 → 월급을 높이더라도

## 세번째 비대면 (251215) — 1시간 42분

### 핵심 주제: 환율/자산 보존 + 포지셔닝 + 욕구술사 + 상징성 + 컴플렉스 마케팅

---

### 1. 환율과 자산 보존 (구체적 수치)
- 엠군 본인: 소득 + 빌딩(150억) + 아파트(50억) + 현금(달러 전환)
- 170만 달러 MMF (연 3.65%, 언제든 출금 가능)
- 1459원에 매수 → 1490원 시 약 1억 환차익 (2주 만에)
- 1600원 시 3.8억, 2000원 시 15억 차익 예상
- **원화로 40억 보유 vs 달러 전환**: 2000원 시 원화 실질가치 24억으로 하락 → 30억+ 차이
- 은이 금보다 더 많이 올랐음 (100%+)

### 2. 포지셔닝 미션 심화
- CV 핵심가치는 2개까지만 (인간의 기억 한계)
- **후추 스프레이 예시**: 살상력(1순위) + 가격(2순위) → 제조국/분사력은 의미 없음
- 살상력 높이고 가격 프리미엄 = 엠타트업식 기획력
- 관여도 높은 호신용품 → 5만원이 아니라 50만원까지 가능

### 3. 욕구술사 = 단체주문 사고
- 치킨집 시절: 회식 키워드 → 페이백(도장 적립) → 회식 매출 폭발
- **B2B가 아닌 "말단 직원 결정권자 타겟팅"**
- 단체주문 가능 제품을 기획하면 마진+효율 폭발

### 4. 컴플렉스 = 돈
- 본인만 알고 남한테 말 못하는 컴플렉스 → 가장 깊은 결핍
- **미션**: 타겟의 컴플렉스 5개 → 각각에 맞는 R 접근 = 부자가 되는 길
- 엠군 본인 컴플렉스: 마른 체형 (주변은 몰랐음)

### 5. 상징성 싸움
- 온라인 시장 = 상징성(브랜드인 척) 싸움
- 지하상가/보조브랜드가 브랜드인 척하는 방법 → 우리도 동일
- **셀룰라이트 시장**: 블루오션, 허브키워드 없음, 결핍 깊음, 상징성 강조 후 진입 권장

### 6. 스토리텔링 상세페이지 오프닝
- 관여도 6+ (고가 제품): 판매자 본인의 컴플렉스 스토리로 대입 → "나도 그랬는데" → 동화 → 다음 내용 읽게 됨
- 공보(피부) 예시: 판매자가 공보였던 스토리 → 타겟 공감 → 솔루션 제시

## 마지막 비대면 (251231) — 1시간 51분

### 핵심 주제: 주체성 + 배제 심화 + 즉각 실행 + 욕구 > R(알) + 캔디 시장 + 자연적R

---

### 1. 연말 분위기와 주체성
- 31일이 중요한 사람 = 인생에서 나이 외에 내세울 CV가 없는 사람
- 내 인생 스케줄이 중요한 사람에게 연말은 의미 없음
- **자산에 0 하나 붙이는 것 >> 나이에 1 더하는 것**

### 2. 배제 심화 (귤 비유)
- 썩은 귤을 옆에 두면 멀쩡한 귤도 맛이 변함
- **배제할 2유형**: (1) 같이 있으면 즐겁지만 인생 낭비되는 사람 (2) 우월감은 느끼지만 에너지 빨리는 사람 → 둘 다 배제
- 엠군: 10년 넘게 술자리/모임 안 나감 → "정제된 인생"

### 3. 즉각 실행 = "따닥이"
- 생각하면 두려워짐 → 생각 전에 실행
- 돈벌레 비유: 3미터 커지기 전에 즉시 잡아야 함
- **"어렵다"는 말 금지**: 지하철 타는 것도 처음엔 어려웠음, 이커머스는 효율적이고 쉬운 편

### 4. 욕구가 R(알)보다 상위 개념 (핵심!)
- **청소 예시**: 청소 → 청소기? NO. 청소하고 싶은 심리 → 여친에게 잘 보이고 싶음 → 페로몬 향수/탈취제가 될 수도 있음
- 고인돌 시대에도 청소 심리는 존재 → R(알)은 시대마다 다름 → **욕구에서 출발하면 R은 무한**
- 여러분의 고정관념이 R을 가두고 있음 → 욕구깡패가 되려면 R부터 생각하면 안 됨

### 5. 캔디 시장 = 접근성의 무기
- 캔디라는 형태의 간편함 → 다양한 결핍에 적용 가능
- 숙취캔디, 피로회복캔디, 키크는캔디, 탈모캔디, 정력캔디, 비염캔디...
- 허브키워드 만들기 쉬움 + 소비자 접근성 최고

### 6. 일시적R과 오감 활용
- 통증 = 효과의 증거 (리프팅이 아프면 잘하는 성형외과)
- "아픈 사랑 = 진짜 사랑" → 통증을 혜택으로 전환
- 촉감/따끔함/가려움 = 일시적R의 핵심 → GPT에 오감 촉감 예시 전부 뽑아서 활용

### 7. 셀룰라이트 시장 (재언급)
- 여전히 블루오션, 제대로 된 브랜드 없음
- 상징성 + 브랜드 네이밍 먼저 → 기능은 5/10 수준이면 충분

### 8. 본인 궤도 찾기
- 엠군 2025년 = 방어회(수익 지키기), 2026년 = 공격회(더블업)
- 무리하면 고장남 → 우울증약 먹는 교육생 10명+ 경험
- 자기 템포 찾아서 업다운 관리해야 함

## 전체 에렌 적용 포인트 종합

1. **식물미스트**: 욕구 형상화 + "당신 잘못이 아닙니다" 적용. 식물 죽이는 건 당신 탓이 아님 → 케어 방법을 몰랐을 뿐
2. **분갈이세트**: 컴플렉스 마케팅 — 초보 식집사의 "실패할까봐" 불안 → "이 세트면 실패 불가"
3. **무료세미나**: 영수화법 + 적의적 프레임. "월급만으로 안 되는 건 당신 잘못이 아닙니다" → 편들어주기 → 솔루션 제시
4. **진격의 클래스**: 스토리텔링 오프닝 — 에렌의 야구선수 부상→비닐하우스→연매출23억 스토리 = 완벽한 대입 소재
5. **마케팅 대행사**: 단체주문 사고 적용 — 개인 셀러 1건씩이 아닌, 업체 단위 패키지
6. **투자**: 원화 → 달러 전환 고려, 하급지 부동산보다 유동성 높은 자산
7. **배제**: 진격의 클래스 + 마케팅 대행사에 집중, 나머지 줄이기
8. **즉각 실행**: 생각 전에 테스트 → "따닥이" 마인드
