# 엠타트업 21기 강의 핵심 정리

> 멘토: 엠군 (엠타트업 대표)
> 코치: 라온, 작은영화, 캐리아
> 기간: 8주 (W1~W8, W9는 비대면)

---

## W1: 오프닝 + 사고 스트레칭 + 목표 설정

### 오프닝 (캐리아)
- 21기 성장 시스템 소개: 주변환경조성, 미행자 시스템(미션 불이행시 벌칙), 감시탑(웹캠 상호감시), 자리상승제도(5주차부터 경쟁), 1:1 톡방
- 캐리아의 역할: 1,700명 상담 경험, 700명 1:1 피드백 → 코칭 전문
- 21기 특징: 월천 이상 수익자 70%+ → 역대 최고 평균 수익

### 엠군 강의
- **프레임 스트레칭**: 돈/가족/키 등 기존 고정관념 깨기 연습
  - 돈이 많다고 행복한 게 아님 → "누구한테 어느 정도"가 중요
  - 가족은 무조건 소중? → 주체성 없으면 가장 큰 가해자가 됨
  - 프레임을 깨고 → 부족한 기능을 장점으로 전환하는 훈련
- **실전 예시**: 손잡이 없는 실내자전거 → 프레임 깨기로 대히트
  - 기능을 빼고, 그걸 장점으로 포장 (저출력 레이저 등)
- **부동산 목표 설정**:
  - 상급지(상급지) 로그인 = 최소 15억(15개)
  - 21기 목표: 21개를 가장 빨리 모으는 과정
  - 포르쉐/사치품 금지 → 15개 모을 때까지 모닝도 사치
  - "반지키스 금지" = 아직 꿀도 안 먹었는데 세레머니하지 마라
- **카테고리 대장 전략**: 큰 카테고리에서 1등 → 한방 터뜨려서 시드 모으기
- **몰입 방법**: 배제할 것들 - 화장/친구/유튜브/게임/술/연애/감정/네이버뉴스

## W2: 주체성 + 타겟 마케팅 + AI 활용

### 라온 강의
- **주체성이 핵심**: 기회가 왔을 때 잡을 수 있는 건 주체성 있는 사람뿐
  - 라온 본인의 아파트 매수 기회 놓친 사례 → 주체성 없어서
  - 작은영화(자켓)님은 바로 잡음 → 격차 벌어짐
  - "깨뚫림 없는 주체성은 위험" → 사고력 + 주체성 함께 필요
- **시대 변화 캐치**:
  - 물길(트래픽) 작성: 사람 → AI(GPT)로 전환 → 무한 양산 가능
  - 채용 방식: 사무실 직원 → 재택 근무자 → 고정비 절감 (연 0.6개 세이브)
  - AI 이미지/영상 → 연령 높은 타겟에게 특히 효과적
- **타겟이 전부다**:
  - 낚시 비유: 볼락엔 지렁이, 고등어엔 크릴새우 → 타겟이 원하는 말을 해줘야 함
  - 여드름 예시: 피부과 못 가는 타겟(10대, 임산부, 군인) → 교환이 더 쉬움
  - 귀마개 하나로 3개 시장: 청각소음/군대/수능 → 동일 제품, 다른 메시지
  - 군대 귀마개 → 실 구매자는 부모님 → "엄마, 사격하고 저만 괜찮았어요"
  - 수영 샴푸 예시: 염소 제거 → 수영하는 30~50대 여성 타겟
  - **검색량이 아니라 타겟부터 생각하라**

## W3: 부동산 실전 + 매복(마케팅 심화)

### 부동산 파트
- 미분양 → 폭등 패턴: 타워팰리스, 갤러리아포레, 아크로서울포레스트, 나인원한남 등 모두 초기 미분양
  - 공통점: 시대 대비 높은 분양가 → 미분양 → 결국 2~5배 상승
  - "당신이 이 시기에 살 수 있는 가장 비싼 걸 사고, 버텨라"
- 파크사이드(용산 UN3부지) 신규 개발 소개
- 4가지 선택지: ①재건축 매수 ②가라타기(계속 갈아탐) ③풀레버리지 ④수도권 살기좋은곳

### 김솔지 대표 발표 (졸업반 1호)
- 13기 수료 → 5년간 40억 모음 → 아크로삼성 52억 매수
- **솔직 공식**: 나의 결핍 × 높은 가격 = 오래 해먹는 제품
  - 발각질 제거 제품: 시장가 2~3만원 → 12.9만원으로 6배 인상
  - 주변 반대에도 주체적으로 진행 → 카테고리 대장
- 스테디 2개 + 시즌 1개 = 40억 (개인사업자)
- 매수 과정: 스피드가 핵심, 이틀 이상 고민 안 함
- 엠군이 옵션비 1,500만원 거부 협상, 계약서 중도금 변경 → 파기 방지 장치

## W4: 현상의 이해 + 필터링 + 관여도

### 라온 강의
- **방향이 중요**: 직장인 투자법 vs 엠타 투자법은 완전히 다름
  - 직장인: 소액 안전 투자 / 엠타: 풀레버리지 + 가장 비싼 곳
  - 엠타 방식: 가치(가격)를 높여서 마케팅 → 유통(가격 경쟁)과 정반대
- **프레임 깨기**: "부동산 폭락한다" 류의 프레임에 휘둘리지 말 것
- **과녁(타겟) 강조**: 이전 주차 복습 + 실행 독려
- 원베일리 임장 → 동기부여 (계급이 다르다는 체감)

### 엠군 강의
- **필터링 3가지**: 배제 / 공격 / 여과
  - 여과: 나한테 순환되는 사람은 지키고, 해로운 것은 걸러내기
- **제로의 심리 5가지**: 당위성 / 결핍성 / 기발성 / 시급성 / 비교성
- **관여도**: 중요도는 올리고 / 고민(스크)는 제거하기
  - 중요도 올리기: 시간 계급, 미래 지향, 자존심 자극, 희소성, 결핍 강조
  - 고민 낮추기: 절차 단순화, 혜택 강조, 불필요한 비교 제거

## W5: 중간점검 + 감정 도둑 + 제로의 심리 실전

### 캐리아 강의
- **감정 도둑 대응**: 700명 1:1 케어 경험에서 나온 노하우
  - 감정에 도둑당하면 매출도 멘탈도 무너짐
  - 대응법: 캐릭터 분리 (감정 공감 ≠ 본체 에너지 소모)
  - "딱 1번 필터 끼고" → 공감하되 본체는 영향 안 받게
- **제로의 심리 실전 적용**: 멘탈 나간 교육생 행동 변화시키기
  - 당위성: "다 멘탈 나간다, 해야 할 건 정해져 있어"
  - 결핍성: "1주일 안 하면 2,100만원 손해"
  - 기발성: "헤어졌으면 꿀버커싱으로 소개팅 잡자"
  - 시급성: "상급지 숫자 줄고 있다, 사다리 끊기 전에"
  - 비교성: "같이 시작한 교육생 벌써 월천"
- **관여도 실전**: 중요도 올리고 고민 제거하는 1:1 상담 예시

## W6: 라온의 실전 사례 + 욕구/타겟 심화

### 라온 강의
- "고소리 서말이라도 꿰어야 고배다" → 배운 걸 실행에 연결
- **일상에서 사업 기회 발견**: 어머니 이사 중 하수구 막힘 → 셀프 뚫는 도구 시장 발견
  - 하수구 뚫기 도구: 10.7만원, 하루 2개씩 판매 → 나쁘지 않은 마진
  - 업자 부르면 20~45만원 → 셀프 도구가 훨씬 저렴
  - 지역 키워드(대전 싱크대, 전주 하수구 등) → 경쟁 거의 없음
- **욕구와 타겟에서 항상 출발**: 구축 아파트 거주자, 주택, 공장 등
- 물길 실전: 카페, 블로그, 지역 키워드 활용

## W7: 채널(C) 전략 + 작은영화 코치의 12개 모은 법

### 작은영화 코치 강의
- **RCS 중 C(채널)이 핵심**: 틱톡을 무기로 활용
- **포젝스(42억 아파트) 매수 스토리**:
  - 라온이 포기한 물건을 잡음 → 1년간 월 5,300만원씩 적금 필요
  - 실제로는 11억 이상 필요 (이자, 취득세, 이사비, 법인→개인 세금 등)
  - 16년간 10개 모은 돈 vs 1년간 12개 모음 → "제약이 동력"
- **채널 전략**: 식품 5개 런칭, 원가 2,000~2,500원 → 판매가 3만원대
  - 옵션 설계로 객단가 5만원 → 마진 90%
  - 1~2월 매출 기록적 → 택배 1만개 이상
- **씬(Scene)은 끈기**: 2017년 페이스북 광고 → 동영상 편집 학습 → 틱톡 진입 시 즉시 실행 가능
  - 과거의 쌓임이 새 채널 진입 시 무기가 됨

## W8: 마지막 강의 + 물길 파트 + 자각/관찰

### 엠군 강의
- **부동산 업데이트**: 21기 중 13명 상급지 거래 완료
  - 운영진 100%, 1조 60% 도장
  - "나는 죽으면 같이 죽어" → 먼저 하고 같이 하자
  - 내년 서울 신축 4,158가구뿐 (역대 최소) → 지금이 기회
- **자각과 관찰이 인생 치트키**:
  - 해보지 않으면 모른다 → 거주환경 바꿔보기, 차 타보기
  - 관찰이 되면 남이 힘들게 얻는 걸 쉽게 파악 가능
- **빈배 필터 (장자)**: 부딪히는 배에는 아무도 없다
  - 나를 공격하려고 한 게 아니라 그냥 흐름에 있던 것
  - 화날 때 "빈배 필터 껴" → 감정 소모 방지
- **반지키스 필터**: 아직 성과 없는데 세레머니하지 마라
- **전쟁 공포 마케팅**: 시즌/이슈 활용한 물길 전략
  - 인스타 전쟁 콘텐츠 → 댓글로 제품 연결 → 자연스러운 물길
- **통찰의 중요성**: 자각 → 관찰 → 인지 단계
  - 21기 게시판의 노하우 하나가 다른 사람에겐 평생 넘을 수 없는 벽

## 전체 핵심 키워드

| 개념 | 설명 |
|------|------|
| 프레임 깨기 | 기존 상식을 뒤집어 새로운 가치 창출 |
| 주체성 | 모든 선택의 근본, 없으면 돈 벌어도 의미 없음 |
| 타겟 | "타겟이 전부다" - 타겟이 원하는 말을 해줘야 교환 |
| 깨뚫림 | 시대 흐름/본질을 꿰뚫어 보는 능력 |
| 필터링 | 배제/공격/여과 → 에너지 도둑 차단 |
| 제로의 심리 | 당위성/결핍성/기발성/시급성/비교성 |
| 관여도 | 중요도 ↑ + 고민(스크) ↓ |
| 솔직 공식 | 나의 결핍 × 높은 가격 = 장기 수익 |
| RCS | Resource(자원) / Channel(채널) / Scene(장면) |
| 반지키스 금지 | 성과 전 사치 금지, 15개 이상 모을 때까지 |
| 빈배 필터 | 부딪히는 상대에 악의 없음 → 감정 소모 방지 |
| 카테고리 대장 | 1등만이 상급지 로그인 가능 |
