2. 백엔드 요구사항 (개발 범위)

A. 모임 등록/관리 (호스트)

기능설명우선순위
모임 생성제목, 설명, 카테고리, 날짜/시간, 장소, 정원, 모임비필수
모임 수정/삭제호스트가 본인 모임 편집필수
카테고리 분류재테크/사업/운동/연애/맛집/자기계발 (6개 + 하위)필수
모임 상태 관리모집중/마감/완료/취소필수
반복 모임정기 모임 자동 생성2차

B. 참가 신청 처리

기능설명우선순위
신청참가자가 모임 신청 (이름, 연락처, 간단 소개)필수
승인/거절호스트가 신청자 승인 또는 거절필수
대기열정원 초과 시 대기 리스트필수
노쇼 관리노쇼 기록, 반복 노쇼 제한2차
알림신청/승인/거절/리마인드 알림필수

C. 결제 연동

기능설명우선순위
모임비 결제토스페이먼츠 연동, 신청 시 결제필수
환불 처리취소 시 자동/수동 환불필수
정산호스트에게 모임비 정산 (수수료 제외)필수
무료 모임결제 없이 신청만필수

D. 카톡/오픈채팅 연동

기능설명우선순위
오픈채팅 링크모임별 카톡 오픈채팅방 링크 자동 노출필수
승인 후 공개참가 승인된 사람만 링크 열람필수
카톡 알림알림톡 API 연동 (신청/승인/리마인드)2차

E. 어드민 패널

기능설명우선순위
회원 관리회원 목록, 검색, 차단필수
모임 관리전체 모임 조회, 강제 취소필수
정산 관리호스트별 정산 현황, 수수료 관리필수
통계 대시보드가입자 수, 모임 수, 매출, 카테고리별 현황2차
신고 관리부적절 모임/회원 신고 처리2차

F. 회원/인증

기능설명우선순위
회원가입/로그인카카오 소셜로그인 (우선), 네이버/구글필수
프로필이름, 사진, 소개, 관심 카테고리필수
호스트 신청일반→호스트 전환 신청/승인필수
