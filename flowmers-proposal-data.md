## 2. 결정사(결혼정보회사) 시장 분석

### 시장 규모
- 국내 결혼정보회사 시장: 약 3,000~4,000억원 규모 (추정)
- 1인가구 증가 + 만혼 트렌드 → 매칭 서비스 수요 지속 성장
- 소개팅 앱 시장 포함 시 1조원 이상

### 기존 시장의 문제점
| 문제 | 설명 |
|------|------|
| 높은 마케팅 비용 | 듀오, 가연 등 대형사 TV/온라인 광고에 수십억 투자 |
| 신뢰도 이슈 | 회원 불만, 환불 분쟁 빈번 |
| 인력 의존적 | 커플매니저 채용·교육 비용 높음 |
| 획일적 서비스 | 가격대 높고, 젊은 층 접근성 낮음 |

### 플로우머스의 기회
- **마케팅 비용 혁신:** 자체 마케팅 노하우로 CPA(가입당 비용) 획기적 절감
- **인력 확보 혁신:** 강의 → 프리랜서 매니저 파이프라인 (채용비용 0)
- **콘텐츠 차별화:** 유튜브·블로그 기반 신뢰 구축

## 3. 차희연 대표 협업 구조

### 차희연 대표 포지셔닝
- **강사 / 크리에이터:** 커플매니저 양성 교육의 메인 강사
- 결정사 시장 자체보다 **강의/브랜딩** 기회에 관심
- 본인의 전문성을 활용한 교육 콘텐츠 제작

### 역할 분담
| 구분 | 플로우머스 | 차희연 대표 |
|------|-----------|-----------|
| 마케팅/DB 수집 | ✅ 전담 | - |
| 광고비 투자 | ✅ 전액 부담 | - |
| 강의 기획/운영 | ✅ 공동 | ✅ 메인 강사 |
| 커리큘럼 개발 | 지원 | ✅ 주도 |
| 랜딩페이지 제작 | ✅ 전담 | - |
| 수강생 관리 | ✅ 시스템 | ✅ 교육 |

### 수익 배분
- **플로우머스 60% : 차희연 대표 40%**
- 광고비는 플로우머스 측 전액 부담
- 차희연 대표는 강의 진행만으로 수익 확보 (리스크 제로)

## 4. 사업 모델: 강의 → 인력 확보 파이프라인

### 전체 플로우
```
[1단계] 마케팅/광고
    ↓ (메타광고, 당근비즈프로필, 네이버카페 등)
[2단계] DB 수집 → 수강생 모집
    ↓
[3단계] 차희연 대표의 "커플매니저 부업" 강의
    ↓
[4단계] 교육 수료
    ↓
[5단계-A] 강의 매출 발생 (즉시 수익)
[5단계-B] 수료생 → 플로우머스 프리랜서 매니저 계약 (인력 확보)
    ↓
[6단계] 프리랜서 매니저가 결정사 매칭 업무 수행
    ↓
[7단계] 결정사 서비스 매출 발생 (장기 수익)
```

### 이 구조의 5대 장점
1. **즉시 매출:** 강의 판매 자체가 수익 (교육비 수입)
2. **인력 비용 제로:** 채용 공고·면접·교육비 없이 훈련된 인력 확보
3. **스케일러블:** 강의 회차 반복 → 인력풀 자동 확대
4. **저리스크:** 광고비만 투자, 강사 고정급 없음 (성과 배분)
5. **충성도:** 교육받은 수료생 = 브랜드에 대한 이해도 높은 인력

## 7. 마케팅 전략

### 수강생 모집 채널
| 채널 | 전략 | 예상 CPA |
|------|------|---------|
| 메타(인스타/페북) 광고 | 부업 관심 여성 타겟팅 | 3~5만원 |
| 당근 비즈프로필 | 지역 기반 무료세미나 모집 | 1~3만원 |
| 네이버 카페 바이럴 | 육아맘/부업 카페 글 자동화 | 거의 0원 |
| 유튜브 콘텐츠 | 오맷돌 대표 영상 트래픽 | 0원 (오가닉) |

### 전환 퍼널
```
광고 노출 → 무료세미나 신청(리드) → 세미나 참석 → 유료 강의 전환
```
- 무료세미나로 진입장벽 낮춤
- 세미나에서 차희연 대표의 전문성 어필 → 유료 전환

## 8. 차희연 대표에게 드리는 가치

### 강사로서의 이점
1. **마케팅 부담 제로:** 수강생 모집은 플로우머스가 전담
2. **광고비 부담 제로:** 전액 플로우머스 투자
3. **브랜딩 기회:** 강의를 통한 개인 브랜드 구축
4. **안정적 수입:** 수강생 모집만 되면 40% 수익 보장
5. **콘텐츠 자산:** 강의 영상·자료 = 본인 포트폴리오

### 리스크
- **차희연 대표 리스크: 거의 없음** (시간 투자만, 금전 투자 없음)
- 플로우머스가 광고비·시스템·마케팅 전부 부담
