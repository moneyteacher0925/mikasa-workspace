# 무농약상토 유튜버 협찬 가이드

> 작성일: 2026년 3월 24일 | 제품: 무농약상토 | 타겟: 가족 웰빙·건강 원예 라이프

---

## 1. 유튜버 선정 기준

| 조건 | 기준 | 이유 |
|------|------|------|
| 구독자 수 | 1만~10만 (소형~중형) | 단가 낮고 참여율 높음 |
| 카테고리 | 원예/가드닝/텃밭/살림/육아 | 타겟 시청자 보유 채널 |
| 평균 조회수 | 최근 10개 영상 평균 3,000회 이상 | 실제 시청자 활동 확인 |
| 참여율 | 댓글·좋아요 비율 높은 채널 | 전환율 높은 채널 선별 |
| 콘텐츠 톤 | 가족친화적, 건강/웰빙 지향 | 타겟과 일치 |
| 최근 활동 | 최근 1개월 내 업로드 있는 채널 | 휴면 채널 제외 |

---

## 2. 협찬 조건 (우리가 제시할 내용)

| 항목 | 내용 |
|------|------|
| 제공 방식 | 제품 무료 제공 (무농약상토 + 부자재 세트) |
| 보수 옵션 ① | 제품만 제공 (소형 채널) |
| 보수 옵션 ② | 제품 + 소정의 광고비 (중형 채널) |
| 보수 옵션 ③ | 제품 + 수익쉐어/성과형 (협의) |
| 콘텐츠 형태 | 리뷰 영상 / 분갈이 브이로그 / 텃밭 만들기 등 |
| 필수 포함 내용 | 제품명(무농약상토), 구매링크, "무농약" 강조 멘트 |
| 게시 기한 | 제품 수령 후 2~3주 이내 업로드 |
| 영상 유지 기간 | 최소 6개월 삭제 금지 |
| 2차 활용 | 우리 채널/SNS에서 영상 캡처·공유 허용 여부 협의 |
| 수정 권한 | 자유 제작 권장 (자연스러움이 전환율 높음) |

---

## 3. 성과 측정 기준

| 지표 | 측정 방법 |
|------|-----------|
| 조회수 | 영상별 추적 |
| 유입 클릭 | 전용 추적링크 또는 쿠폰코드 (유튜버별 별도) |
| 전환 매출 | 쿠폰코드별 매출 집계 |
| CPV (조회당 비용) | 총 비용 ÷ 조회수 |
| ROAS | 매출 ÷ 협찬 비용 |

---

## 4. 추천 타겟 채널 유형

| 채널 유형 | 예시 콘텐츠 |
|-----------|-------------|
| 베란다 텃밭 / 주말농장 | 텃밭 가꾸기, 작물 재배 |
| 식물 키우기 / 가드닝 브이로그 | 분갈이, 식물 관리 팁 |
| 육아맘 살림 채널 | 아이와 함께 텃밭 체험 |
| 귀농·귀촌 라이프 | 시골 생활, 농사 일상 |
| 친환경/웰빙 라이프스타일 | 건강한 생활, 유기농 |

## 5. 결정 필요 사항

| 항목 | 선택지 |
|------|--------|
| 1건당 예산 범위 | 제품만 / 제품+10만 / 제품+30만 / 기타 |
| 동시 협찬 수량 | 3명 / 5명 / 10명 / 기타 |
| 보수 방식 | 제품만 / 광고비 포함 / 수익쉐어 |

→ 위 3가지 정해주시면 바로 유튜버 후보 리스트 뽑아드립니다.
