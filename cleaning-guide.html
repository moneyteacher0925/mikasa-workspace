  • 소모품 부족 시 아르민 실장님께 즉시 보고

🔄 청소 주기 요약

주기항목

🚻 1. 내부 화장실 매일

순서항목방법
1변기 내부변기 세정제 도포 → 솔로 꼼꼼히 문지르기 → 물 내리기
2변기 외부 / 시트세정제 + 걸레로 닦기 (시트 양면 모두)
3세면대세정제로 물때 제거 + 수전 닦기 + 물기 제거
4거울유리세정제 분사 → 마른 천으로 결 없이 닦기
5바닥쓸기 → 물걸레 (배수구 이물질 제거)
6쓰레기통비우기 + 봉투 교체
7소모품 보충휴지, 핸드타올, 비누 잔량 확인 → 보충
8문 손잡이세정제로 닦기

주 1회

환풍기/환기팬 외부 닦기
타일 줄눈 곰팡이 확인 → 발견 시 보고

월 1회

환풍기 필터 분리 청소
배수구 집중 청소 (세정제 + 뜨거운 물)

🚾 2. 외부 화장실 매일

순서항목방법
1변기 내부변기 세정제 + 솔
2변기 외부 / 시트세정제 + 걸레
3세면대세정제 + 물기 닦기
4거울유리세정제 + 마른 천
5바닥쓸기 → 물걸레
6배수구머리카락/이물질 제거
7쓰레기통비우기 + 봉투 교체
8소모품 보충휴지, 핸드타올, 비누
9문 손잡이세정제로 닦기

※ 주 1회 / 월 1회 항목은 내부 화장실과 동일

☕ 3. 휴게실 매일

순서항목방법
1식탁 / 의자행주로 닦기 (의자 등받이 포함)
2싱크대 / 조리대세제 + 수세미 → 물기 완전 제거
3전자레인지내/외부 젖은 행주로 닦기 (음식물 찌꺼기 제거)
4정수기 외부마른 천으로 닦기
5바닥쓸기 → 물걸레
6쓰레기통비우기 + 봉투 교체
7분리수거정리 / 배출

주 1회

냉장고 외부 닦기
수납장/선반 먼지 제거

월 1회

냉장고 내부 정리 (유통기한 지난 것 폐기)

🖥 4. 사무실 매일

순서항목방법
1바닥쓸기 → 물걸레 (또는 밀대)
2공용 테이블마른 천으로 닦기
3쓰레기통비우기 + 봉투 교체
⚠ 사무실 주의사항
  • 개인 책상 위 서류/장비 절대 이동 금지
  • 개인 책상 상판은 닦지 않음 (본인이 비워둔 경우만)
  • 복합기/프린터 — 외부만 닦기 (버튼 부분 주의)
  • 모니터 화면 닦지 않기 (전용 클리너 필요)

주 1회

창문턱 / 선반 먼지 제거
공용 장비 (복합기, 정수기) 외부 닦기
출입문 / 손잡이 닦기

월 1회

에어컨 필터 확인 (이상 시 보고)
구석 / 책상 뒤 먼지 제거
