```

2. 수집 목적:
   - 모임 운영 및 안내 (참가 확인, 입금 확인, 오픈채팅방 안내)
   - 맞춤형 모임 추천 및 마케팅 정보 제공
   - 서비스 개선을 위한 통계 분석
3. 보유 기간: 수집일로부터 1년 (목적 달성 후 지체 없이 파기)
4. 동의를 거부할 권리가 있으나, 거부 시 모임 신청이 불가합니다.
주식회사 플로우머스
```
선택지: "위 개인정보 수집 및 이용에 동의합니다."

## 카테고리별 고유 항목

### 1. 재테크
**폼 제목:** [딱모여] 재테크 모임 신청서
**폼 설명:** 💰 같은 관심사를 가진 사람들과 함께 재테크 이야기를 나누는 오프라인 모임입니다.

| 항목 | 유형 | 필수 | 선택지 |
|------|------|------|--------|
| 관심 재테크 분야 | 체크박스(복수) | O | 주식 (국내/해외) / 부동산 (투자/임대) / 코인 (암호화폐) / 기타 (예: 금 투자, ISA 계좌 등) |
| 투자 경험 | 객관식 | O | 아직 시작 전 / 1년 미만 / 1~3년 / 3년 이상 |
| 투자 가능 자금 | 객관식 | O | 100만원 미만 / 100~500만원 / 500~1,000만원 / 1,000~5,000만원 / 5,000만원 이상 / 밝히고 싶지 않음 |

### 2. 사업

**폼 설명:** 🚀 사업가, 예비 창업자들이 모여 경험과 인사이트를 나누는 오프라인 모임입니다.

| 항목 | 유형 | 필수 | 선택지 |
|------|------|------|--------|
| 관심 사업 분야 | 체크박스(복수) | O | 스타트업 / 이커머스 / 1인사업가 / 마케터 / AI / 요식업 / 자영업자 / 기타 |
| 현재 사업 상태 | 객관식 | O | 준비 중 (아직 시작 전) / 1년 미만 (초기) / 1~3년 (성장 중) / 3년 이상 (안정기) / 관심만 있어요 |
| 월 매출 규모 | 객관식 | O | 아직 매출 없음 / 500만원 미만 / 500~3,000만원 / 3,000만원~1억 / 1억 이상 / 밝히고 싶지 않음 |
