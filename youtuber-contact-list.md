## 이메일 확인 방법 (미확인 채널)

1. 각 채널 URL 접속
2. **정보** 탭 클릭
3. **비즈니스 문의** → **이메일 보기** 버튼 클릭
4. 이메일이 없는 경우 → 댓글/커뮤니티 탭에서 DM 또는 블로그/인스타로 연락

> 💡 유튜브 비즈니스 이메일은 로그인 상태 + JS 렌더링이 필요하여 자동 수집이 제한됩니다.
